Description

With this object, you can make your program a complete ShareWare. He will take all the worries about tracking the trial period for the activation of the key registration, etc.

Trial will make an automatic registration program. Example of use of this object is in the folder Examples \ Complex projects \ Trial

object does not give any guarantee that your ShareWare program jeopardized. Trick for the attacker is not a particularly large amount of labor. Therefore, we recommend to combine object Trial, with its methods of tracking the trial period, and activation. In order to complicate the work of the cracker.

Remark

Each object is associated with your program only on property Id program (only the unique identifier of your program) and property encryption key (key, indicating the encryption algorithm). Therefore, if you use multiple objects Trial and want them to work with some keys and Id, it follows that Id Program and encryption key they were the same !
