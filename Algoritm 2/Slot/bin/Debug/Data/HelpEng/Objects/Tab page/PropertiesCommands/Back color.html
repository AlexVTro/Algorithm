Description

The main color in which the colored object.

All colors can be specified verbally symbol colors (red, blue, black), more on this topic Auxiliary words -> Colors . Either color can be specified in the special form , where the first is the intensity of red color, then a semicolon intensity of the green, and after the semicolon blue. The maximum intensity of 255, minimum 0. For example: 

"255, 0, 0," = red
 "0, 0, 255; = blue
" 0, 0, 0, "= black
" 255, 255, 255, "= white
 "255, 255, 128," = light yellow

Remark

If you want to make a color transparent (not all objects have this feature), set this property to "No" or "".
