Description

Along with name , Number is insignia property of the object. On this property it is possible to distinguish objects , for example, if they have identical names (in arrays )

The number can be any a positive number. In addition, when you create the action and you want to specify the number of the object, you can use auxiliary word All . In this case, create an action would apply to all objects with the name you specified , regardless of numbers .

Remark

The two objects can not be identical names and both rooms . These two properties are designed to uniquely identify the object in the development environment.
