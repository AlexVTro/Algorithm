Description

The object will be displayed such text , which will be specified in this property.

Example the object Window text is its title, and the object text box is the text of the recorded it.

Remark

If the text you want to add quotes, it becomes a double repetition of the quotes within the text that you want to assign: 
"This is the text without quotation marks, and" here in quotation marks "". "

To insert newline (analog pressing Enter), use Auxiliary words -!> ;! Other , or insert a symbol "¶"

 Window1.Knopka1.Tekst = "text ¶ The text on a new line"
