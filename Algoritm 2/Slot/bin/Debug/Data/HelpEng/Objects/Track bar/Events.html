Description

Events is the user actions (clicks, cursor movement, keystrokes), which can catch objects .

Each object has certain events , which he can catch. Set capture events for objects in Section events . After that, all the actions that you place in this event in Section wood programs , will be performed after the occurrence of the event (eg a click).

CreatedThe event takes place in early of the program. At a time when the object is first created .
Got focusEvent occurs when the object passed focus (made active object)
Key downThe event is now in active object (object in Focus ), at a time when the user presses button on the keyboard. The object on which the event occurred, there are additional properties.
Key pressThe event is now in active object (object in Focus ), at a time when the user clicks button keypad. The object on which the event occurred, there are additional properties.
Key upThe event is now in active object (the object in Focus ), at a time when the user releases the , formerly нажатую button keypad. The object on which the event occurred, there are additional properties.
Lost focusEvent occurs when the active object, took focus (another site has become active)
Mouse downThe event is now in active object (object in Focus ), at a time when the user presses button. The object on which the event occurred, there are additional properties.
Mouse enterThis event fires as soon as the cursor hits the object.
Mouse hoverThis event is constantly until the cursor is located on the site.
Mouse leaveThis event fires as soon as the cursor leaves the object.
Mouse moveThe event takes place at the facility when the user moves on it mouse. The object on which the event occurred, there are additional properties.
Mouse upThe event takes place at the facility when the user releases the , formerly нажатую mouse button. The object on which the event occurred, there are additional properties.
Mouse wheelThe event is now in active object (object in Focus ), at a time when the user starts to rotate the wheel mouse. The object on which the event occurred, there are additional properties.
