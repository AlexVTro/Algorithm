Description

The event is now in active object (object in Focus ), at a time when the user presses button on the keyboard. The object on which the event occurred, there are additional properties.

Additional features which appear at the sites at this event:

Object that happened on the event, adds a happy new features (for example, X and Y mouse, key pressed). These properties can not be changed, but you can see them (because they can not be changed, then find the properties can be Wizard complex action ). Below are all the properties of the event.

Property
Description
_Key pressedContains the name of the keyboard keys (a, b, ctrl, enter), which is pressed during events .
_Keyboard altDisplays, pressed a button keyboard alto (Alt) at the moment or not.
_Keyboard controlDisplays, pressed a button keyboard Control (Ctrl) at the moment or not.
_Keyboard shiftDisplays, pressed a button keyboard Shift (Shift) at the moment or not.
