Description

If in Focus equals "Yes", then the object is active and is now working with the user (eg typing in text box ) .

Concept

Focus has the same meaning as in cameras, ie be in focus means to be in the spotlight. In this case, be the object of your attention. If this property is equal to "No", then the object is still in working condition , just the user probably does not use it yet.
