Description

With this property you can specify the width all columns of the table.

The data required to implement properties or commands:

Objects.Table.Columns width (Column numbers separated by commas)

Data
Description
Column numbers separated by commasHere you want to specify the columns that will be made of. If the column is one, you can ukat so: "0" - the first column, or "2" - the third. If more than one, then: "0,1,3" - the first second and fourth column.
