Description

Properties - a set Objects (color, text size), changing that you can configure the object necessarily. Objects can also perform certain actions on themselves, called command .

can manually change the properties in Properties section . Or program them to change Section Action . In the second case, first for the object is created event in Section events , in which the property should be changed. Examples of properties and commands in Section Action:

Изменять свойства можно вручную в Разделе свойств. Или запрограммировать их изменение в Разделе действий. Во втором случае сначала для объекта создается событие в Разделе событий, при котором данное свойтво должно поменяться. Примеры использования свойств и команд в Разделе действий:

_Cancel pressThis feature allows cancel by the user. This is done change the property value to "Yes".

_Key pressedContains the name of the keyboard keys (a, b, ctrl, enter), which is pressed during events .

_Keyboard controlDisplays, pressed a button keyboard Control (Ctrl) at the moment or not.
_Keyboard shiftDisplays, pressed a button keyboard Shift (Shift) at the moment or not.
_Mouse buttonStores the name of the mouse button (left button, wheel), which made event .
_Mouse clicksContains the number of clicks that are made in this event .
_Mouse deltaContains the number of times as it turned the wheel, in this event .
_Mouse XStores coordinate X cursor at a given time. The property is available, for example, in the event of "clicking", if you choose in the list of objects that object for which the created this event .
_Mouse YStores coordinate Y cursor at a given time. The property is available, for example, in the event of "clicking", if you choose in the list of objects that object for which the created this event .
