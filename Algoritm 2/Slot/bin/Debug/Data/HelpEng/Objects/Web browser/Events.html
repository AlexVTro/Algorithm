Description

Events is the user actions (clicks, cursor movement, keystrokes), which can catch objects .

Each object has certain events , which he can catch. Set capture events for objects in Section events . After that, all the actions that you place in this event in Section wood programs , will be performed after the occurrence of the event (eg a click).

Can go back changedAbility to move Back in the object browser to be present, but may be absent. At the time change this opportunity and triggered the event.
Can go forward changedAbility to move Next in the object browser to be present, but may be absent. At the time change this opportunity and triggered the event.
Document completedThis event fires when the web page completely loaded and displayed by your browser. The object on which the event occurred, there are additional properties.
File downloadThe event is called continuously until the web page is under load in a browser.
Got focusEvent occurs when the object passed focus (made active object)
Lost focusEvent occurs when the active object, took focus (another site has become active)
NavigatedEvent is triggered when a user made follows the link and Upload . The object on which the event occurred, there are additional properties.
NavigatingEvent is triggered before so how will the transition to the new page. The object on which the event occurred, there are additional properties.
New windowEvent occurs when the page wants to open in a new box. The object on which the event occurred, there are additional properties.
Size changedEvent occurs when an object resize (width, the height).
Status text changedStatus bar changes constantly, depending on what is going on in your browser. At every moment change triggered the event.
