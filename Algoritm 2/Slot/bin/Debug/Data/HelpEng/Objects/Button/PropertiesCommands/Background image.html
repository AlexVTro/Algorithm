Background image
Help -> Objects -> Button -> Properties and commands -> Background image

See also, Sample
Collapse all

Description

Figure, which will be located on the background object.

Figure set the path to the image file.

Remark

path to the picture can be specified as absolute , ie beginning with the letter of the hard drive 
"C:/Images/pic.jpg"
and relative . In this case, the path begins to be from the folder in which your project is 
 "/ Images/pic.jp"

Example

Options change the background image in the main window

Aлгоритм2
[Copy to clipboard] 
 Window1.Window1.Fonovoy pattern = "C:/1.bmp" Window1.Window1.Fonovoy pattern = "/ 1.bmp" Window1.Window1.Fonovoy picture = My Documents&"/ 2.jpg" Window1.Window1.Fonovoy image = Window1.Knopka1.Fonovoy picture
