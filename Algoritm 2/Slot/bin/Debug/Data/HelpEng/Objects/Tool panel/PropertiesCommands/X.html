Description

coordinate X upper left corner of the object.

Coordinates on all objects are arranged not as ordinary coordinates. The origin is not in the lower left corner, and in upper left . Thus, the X coordinate is essentially distance from the left edge.

Remark

If you put an object inside another object (such as inside panel ), then the coordinates will be considered on the already this object (panel) .
