Description

Objects (buttons, windows, graphics) are blocks from which all programs are created and

Objects can catch the user (they are called events ), such as for example click , movement of the mouse on the object , change the text , etc. To put the capture of any action, use Section events

Objects can be configured as required by a large number of properties . In addition, properties can be changed at run time by setting the rules of their changes in Section Action . These rules modify the properties and are programming .
