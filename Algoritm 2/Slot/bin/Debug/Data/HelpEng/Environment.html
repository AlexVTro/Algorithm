Description

Development Environment program of Algorithm 2 is a window divided into several parts. Each part is responsible for a certain stage of development program (the creation of graphical user interface, configuration objects, programming).

Scheme development environment:

1 - Pane panel objects allows you to create protagonists programs - Objects (buttons, text boxes, windows).
2 - Section WindowIn Section Window you create the graphic design of your program, having to window Various objects (buttons, text fields, table)
3 - Section properties Section Properties allows you to configure all the properties of the protagonists programs - Objects (buttons, text boxes, windows).
4 - Tree ActionIn Tree Action creates all program stuffing your program. All objects , events , steps , conditions , cycles , comments are in Tree Action
5 - Section Events Events is the user actions (clicks, cursor movement, keystrokes), which can catch objects (buttons, text boxes, windows). Section events allows you to assign objects to those events that will catch them in the process of your program.
6 - Section Action Actions is some program commands ( change the text box , change the color of a button ), that run the program and programmed by you.
7 - Section conditions Terms allows you to specify whether to perform at the moment of action or not. The condition is a rule, usually comparison (for example, text box is "Window1" or width button above 100) and all action that come after this condition will be executed only when a condition is true O (ie, text box is really "Window1" a width button above 100)
8 - Section cycles Cycles is the most tezhe conditions , except that all steps inside the loop will run continuously, while condition remains true (for example, while the height windows above 100)
9 - Section Comments Comments is just your explanatory record in tree action . They do not perform and provide an explanation for you as your actions.

Created file will be your own program and you can do with it what you want: throw off your friends, sell, distribute freely on the Internet.
