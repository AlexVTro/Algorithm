Description

Properties - a set Objects (color, text size), changing that you can configure the object necessarily. Objects can also perform certain actions on themselves, called command .

can manually change the properties in Properties section . Or program them to change Section Action . In the second case, first for the object is created event in Section events , in which the property should be changed. Examples of properties and commands in Section Action:

Изменять свойства можно вручную в Разделе свойств. Или запрограммировать их изменение в Разделе действий. Во втором случае сначала для объекта создается событие в Разделе событий, при котором данное свойтво должно поменяться. Примеры использования свойств и команд в Разделе действий:

Window1.Obekt1.Tekst = "New text1"
