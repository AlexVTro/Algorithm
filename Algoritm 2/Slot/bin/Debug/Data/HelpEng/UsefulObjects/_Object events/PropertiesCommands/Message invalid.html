Description

When a user tries to activate the program, but the registration key does not fit, then it will display message , given in this property.

Remark

If the message you want to add quotes, it becomes a double repetition of the quotes within the text that you want to assign: 
"This is the text without quotation marks, and" here in quotation marks "". "

To insert newline (analog pressing Enter), use Auxiliary words -!> ;! Other , or insert a symbol "¶"

 Window1.Knopka1.Tekst = "text ¶ The text on a new line"
