Description

Property highlights row or shows which lines color . The result is given, for example, in this form: 0,2,3 . Ie Highlight first , third and fourth string (because the numbering of rows is 0, 0, - the first line, 1 - second and t . etc.)

To select, for example, a block of cells should combine this command with the commando numbers of selected columns : 

Window1.Tablitsa1.Nomera selected lines = "1,0"
 Window1.Tablitsa1.Nomera selected columns = "1"
This snippet provides first and second box in the second column.
