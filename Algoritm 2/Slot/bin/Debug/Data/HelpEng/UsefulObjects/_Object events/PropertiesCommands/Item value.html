Description

Property allows you to learn content of any table cell, or change it.

The data required to implement properties or commands:

Objects._Object events.Item value (Line numbers separated by commas, Column numbers separated by commas)

Data
Description
Line numbers separated by commasList the rows, separated by commas within double quotes, for example, "0,1,8,3" or "2".
Column numbers separated by commasHere you want to specify the columns that will be made of. If the column is one, you can ukat so: "0" - the first column, or "2" - the third. If more than one, then: "0,1,3" - the first second and fourth column.
