List of object properties _Text:
_CharsThis property returns a character, a serial number which you have transferred property. The first character has the number "1". If the characters in the text is less than the number you gave, this property returns void, ie, ".
_Chars lengthThis property returns the number of characters in the text.
_Comparecommand compares the text and if they are equal returns "0". If not equal, then when the first text on the alphabet above the second, then the command will be equal to -1. If the second text in alphabetical order above the first, then the command returns "1". Thus using this command can, for example, to organize a sort text alphabetically.
_Consistcommand checks for any characters in the source text. If at least one of these characters is present in the text, then the command returns the serial number of the characters in the source text. For example, if the text "yvAay" to look for "pa8", then Komad return "4", ie Reference character "a" in the source text.
_Consist allVerifies that the text was extremely of the characters to convey the command. If at least one character is not the same command will return a "No".
_Consist nocommand checks for any characters in the source text. If any of these characters is present in the text, the command will return "None" if none of the characters are not contained in the text, the command will be equal to the value of "Yes". For example, if the text "yvAay" to look for "pa8", the command returns "No" (ie, the text contains the character "a").
_Delete quotesIf the source is isolated quotes, the command removes those quotes. Also, if you meet two double quotes ( "") within the text, a separate quotation marks, then they will turn into one double quotes ( "" -!> "). For example, the text " Hello, "" Other "" " after the work of this command will become Hello "Friend" .
_Index ofcommand searches for your specified text and returns the number of characters, from whom, is your search text. Register of letters when searching has value, ie letters "A" and "a" are different. If nothing is found not to be, the command will return the value "0".
_Index of ignore casecommand searches for your specified text and returns the number of characters, from whom, is your search text. If nothing is found not to be, the command will return the value "0". Register of letters when searching not has a value, ie letters "A" and "a" are considered equal.
_Index of line
_Index of regularcommand searches the text you specified template regular expression (for example, "Figure 1 \. (jpg | gif | png)") and returns the number of characters, from whom, is your search string. If nothing is found not to be, the command will return the value "0". To learn how to write regular expressions can be read here http://ru.wikipedia.org/wiki/Регулярные_выражения.
_Index of without quotescommand searches for your specified text and returns the number of characters, from whom, is your search text. Register of letters when searching not has a value, ie letters "A" and "a" are considered equal. If during the search in the text to meet a piece of text, isolated quotes, then this piece is passed and the search it is not performed. If nothing is found not to be, the command will return the value "0".
_InsertInserts in any place in the text that you want.
_Is number textChecks whether the text number ( "23", "-12,54", "3,2 + E12").
_Last index of ignore casecommand searches the text that you specify a string starting from the end text and returns the number of characters, from whom, is your search string. Register of letters when searching not has a value, ie letters "A" and "a" are considered equal.
_LineThis property returns a string, a serial number which you have transferred property. The first line has the number "1". If the lines in the text is less than the number that you gave, this property returns void, ie, ".
_Lines count
_Put in quotescommand enters the text referred to it in quotation marks. Also, if the text already there is a double quote ( "), it is converted into two double quotes (" -!> ""). For example, the text Hello "Friend" after the work of this command will become "Hello," "Friend """.
_RemoveRemoves the text of a certain part.
_Replace allIt searches the text all line, similar to the search pattern. And replaces all these lines to the texts referred to as the replacement text.
_Replace oneIt searches the text is first string, which is designated as a search. And replace this line in the text passed as the replacement text.
_SplitAllows any text divided into parts, separated by a character or string. And then returns the part of the text number you specify. For example, if you break the text "Hello, how are you, eh?" apart, indicating separation character ",". That text would break into "hello", "how are you" and "huh?". And if you specify a number of Part 3, you get the text "as well?".
_Split countLets see how many parts can be divided into text, parts of which will be separated by a character or string. For example, if you break the text "Hello, how are you, eh?" apart, indicating separation character ",". That text would break into "hello", "how are you" and "huh?" And the command will return you to the number equal to "3".
_Split without quotesAllows any text divided into parts, separated by a character or string. And then returns the part of the text number you specify. If during the search in the text to meet a piece of text, isolated quotes, then this piece is passed and the search it is not performed.
_Split without quotes countLets see how many parts can be divided into text, parts of which will be separated by a character or string. If during the search in the text to meet a piece of text, isolated quotes, then this piece is passed and the search it is not performed.
_SubstringAllows you to take from the text of any part of it, if you know the serial number of the first character parts and its length. For example, the text "Hello world!" You can get the second word ( "world") indicating that the top piece of the 8th character, and long pieces of 3 characters. In this case, the result of the command will be the word "peace".
_To lowercommand makes all the letters in the text capital , ie translates into lower case ( "A" -!> "a")
_To uppercommand makes all the letters in the text uppercase , ie Uppercase ( "a" -! ">" A ")
_Trim allIt removes from the end of the text and from the beginning of all whitespace characters (spaces, tabs, paragraphs).
_Trim endRemoves from the end of the text, all whitespace characters (spaces, tabs, paragraphs).
_Trim startRemoves from the beginning of all whitespace characters (spaces, tabs, paragraphs).
