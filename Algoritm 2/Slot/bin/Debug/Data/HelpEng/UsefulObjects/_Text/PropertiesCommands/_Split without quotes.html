Description

Allows any text divided into parts, separated by a character or string. And then returns the part of the text number you specify. If during the search in the text to meet a piece of text, isolated quotes, then this piece is passed and the search it is not performed.

For example, if you break the text Hey, how ',' case, and? apart, indicating separation character ",". That text would break into 'hello' , 'how', 'case' and 'and?' . And if you specify a number of Part 2, you get the text 'how', 'case' . Often you need to know how many parts of crashed text. It is available commando number of parts without the quotes .
