Description

Allows you to take from the text of any part of it, if you know the serial number of the first character parts and its length. For example, the text "Hello world!" You can get the second word ( "world") indicating that the top piece of the 8th character, and long pieces of 3 characters. In this case, the result of the command will be the word "peace".

The data required to implement properties or commands:

_Useful objects._Text._Substring (Original text, Character number beginning piece, Length of piece)

Data
Description
Original textThe text of the original, which will be used by the property or a command.
Character number beginning pieceNumber of characters, from which we must begin to take a piece of text. The first character has number "1", second "2", etc.
Length of pieceThe number of characters which will be a long piece of text.
