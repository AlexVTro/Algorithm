Description

This property shows no changes when the file was last accessed. For example, "16/07/2009 0:29:31".

The data required to implement properties or commands:

_Useful objects._Files and paths._Access time (The path to the file or folder, Time)

Data
Description
The path to the file or folderHere you must specify the path to the file or folder. The path should be marked in quotation marks, eg "C: \ 1.jpg" or "\ 2.jpg" (in the second case, the file "2.jpg" must be located in the folder with the project).
TimeIt should specify the time in double quotes, like this "16/07/2009 0:29:31".
