Properties - a set Objects (color, text size), changing that you can configure the object necessarily. Objects can also perform certain actions on themselves, called command .

can manually change the properties in Properties section . Or program them to change Section Action . In the second case, first for the object is created event in Section events , in which the property should be changed. Examples of properties and commands in Section Action:

Изменять свойства можно вручную в Разделе свойств. Или запрограммировать их изменение в Разделе действий. Во втором случае сначала для объекта создается событие в Разделе событий, при котором данное свойтво должно поменяться. Примеры использования свойств и команд в Разделе действий:

List of object properties _Files and paths:
_Access timeThis property shows no changes when the file was last accessed. For example, "16/07/2009 0:29:31".
_Append textThis command appends in the file you specify text or data. If you want to completely overwrite the file, use the commando Save the file
_ArchiveThis property indicates whether the specified archive file in brackets or not.
_CopyThis command copies a file or folder.
_Create directorycommand creates a folder on your computer. And, for example, the folder "C:/a/b/c" will be created successfully, even if it did not exist before, even the folder "C:/a".
_Create timeThis property shows no changes when the file was created. For example, "16/07/2009 0:29:31".
_Decryptcommand decrypts the previously encrypted file. The encrypted file other accounts in Windows and other computers can not read. The user account Windows, which was encrypted file can read it without problems. You also can not decipher the encrypted files of other accounts in Windows.
_DeleteThis command deletes a file or folder forever.
_Edit timeThis property shows or changes the time when the file was last edited. For example, "16/07/2009 0:29:31".
_Encryptcommand encrypts the file so that other Windows accounts and other computers could not read the file. In this case the user Windows account can read this file freely.
_EncryptedThis property indicates whether the file specified in the brackets is encrypted or not. Encrypted files can only view the user account, Windows, in which the files were encrypted. In other accounts, such files will not open.
_Exist fileThis property indicates whether the file on your computer or not.
_Exist pathThis property indicates whether the folder on your computer or not.
_Find fileThis property is looking for files and folders in this directory it. When searching, you can use the search pattern - "*","?","+". For example, "a *. jpg" - find all files with extension jpg, starting with "A" (for example, "abcdef.jpg", "a2.jpg", "a.jpg"). "a? c.exe" - will find all the applications, consisting of three letters beginning with "a" and ending in "s" (for example, "abc.exe", "a2c.exe")
_FolderThis property indicates whether the specified folder path in parentheses or not.
_Get drivesThis property returns a list of files from the specified folder. Files in the list contains a ";" and look like this: "A: /; C: /; D: /".
_Get extensionThis property specifies the path to the file or folder. Example for file "C:/Program Files/File.jpg", this property returns "C:/Program Files".
_Get file nameThis property specifies the name of the file passed to him from the path to a file or folder. Example for file "C:/Program Files/File.jpg", this property returns "File.jpg". For the folder "C:/Program Files/Algorithm 2, this property returns" Algorithm 2 ".
_Get file name without extensionThis property specifies the file name without the extension of the sent him to the path to a file or folder. Example for file "C:/Program Files/File.jpg", this property returns "File".
_Get file sizeThis property specifies the size of the file in bytes.
_Get filesThis property returns a list of files from the specified folder. Files in the list contains a ";" and look like this: "C:/1.jpg; C:/2.bmp; C:/file.exe".
_Get parentThis property specifies the path to the file or folder. For example, the file "C:/Program Files/File.jpg" or folder "C:/Program Files/Algorithm 2, this property returns" C:/Program Files ".
_Get path nameThis property specifies the name of the parent folder for a file or directory. For example, the file "C:/Program Files/File.jpg" or folder "C:/Program Files/Algorithm 2, this property returns the" Program Files ".
_Get pathsThis property returns the list of folders from the specified directory. Folders in the list are listed by ";" and look like this: "C:/Program Files; C:/Windows; C:/Temp".
_Get rootThis property specifies the root - the disc for the given path. Example for file "C:/Program Files/File.jpg", this property returns "C: /".
_HiderThis property indicates whether the file indicated in brackets hidden or not.
_MoveThis command moves a file or folder.
_Not indexerThis property indicates whether the specified file is not in brackets indexed or not.
_Open fileThis property opens the file and gets all of its contents.
_Read onlyAllows you to find out whether a file read-only or can make it so.
_Rename
_Save in fileThis command saves in that you specify the text or data. If the file contains data, they are completely replaced by new ones. If you want to append the file, use the commando Add text
_Save pictureThis command stores the image that you specify.
_Sys
_TempThis property indicates whether the file specified in brackets is temporary or not.
