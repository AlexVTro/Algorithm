Description

Events is the user actions (clicks, cursor movement, keystrokes), which can catch objects (buttons, text boxes, windows). Section events allows you to assign objects to those events that will catch them in the process of your program.

Create Event for an object, you can simply select it from the list of events, and then click Create. All of this is the section events Development Environments Algorithm 2

Thus, if create event Click object Button , then all actions within this event will be executed when clicking on the button:

ie when clicking on the button title text box will change to "Window1".

Remark

events are not only user actions (clicks, text input, resizing) and events occurring within objects. For example, Change the value of memory cell , end playing video file , Stopwatch tiknul , etc.
