Description

Actions is some program commands ( change the text box , change the color of a button ), that run the program and programmed by you.

Create action for an object you can go to the section Action Development Environments Algorithm 2 :

Suffice select an object , who will perform your action and set property which will vary with the object. By setting below new value for the property and clicking Create , you and create the action:

Thus, when the object Button will produce Click , then all actions within this event will be run . In this case, the title text box will change to a "window"

Remark

new values for properties can be not only simple (text, number) but complex (a mathematical formula, the values of the properties of another object). To create complex actions, use Master complex actions (the button with the wand on the right)
