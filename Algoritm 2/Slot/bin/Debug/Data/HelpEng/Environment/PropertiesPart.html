Description

Section Properties allows you to configure all the properties of the protagonists programs - Objects (buttons, text boxes, windows).

Properties is set Objects (color, text size), changing that you can configure the object necessarily. To learn more about the properties of each object you can see Objects .

To change the property, just select it from the list, edit the value and press Enter.
