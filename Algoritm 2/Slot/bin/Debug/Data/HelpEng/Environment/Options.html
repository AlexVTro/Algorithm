Description

Preferences window opens click on the Tools -> Settings . Sections of the Settings window:

Language:
Here you can select the language development environment . In this language will be all the menus and inscriptions, but all actions and programming will continue on the language in which the project was established.

Programming Language:
This is the language which will programmed all new projects , created after changing this setting. All activities and programming current project will continue on the language in which was created project. To change the programming language of the current project, use the Tools -> Translate project on .

Skin:
Theme design development environment Algorithm 2.

the project folder:
folder, which by default creates projects. You can ask here any your folder.

folder with pictures of the project by default:
When your project in the program you section properties specify the drawings, they are automatically copied to special folder pictures , which is located inside folder of your project . Name this folder of drawings and you can change this setting, and all new projects, it will be such as you have. But in the current project she not change .

folder with pictures of the project:
When your project in the program you section properties specify drawings, they are automatically copied to special folder pictures , which is located inside folder of your project . Name this folder of drawings and you can change this setting. And in the current project she change .

name for the image files by default:
When your project in the program you section properties specify the drawings, they are automatically copied to the drawings, which is located inside the folder of your project. This picture name changes to specified in this setting.

Icon program:
When you create a project from its executable exe -file (File -> Create an existing program) it marked the file icon will be presented here.

Show download window when you start the project:
When your program will run user, then before the program finally loaded and displayed in front of him will some time . At this time, will be displayed download window , which run through a strip-light download the program. This setting you can disable this option.
