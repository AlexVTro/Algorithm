Description

Cycles is the most tezhe conditions , except that all steps inside the loop will run continuously, while condition remains true (for example, while the height windows above 100)

Create cycle You can go to the section cycles Development Environments Algorithm 2 :

It should introduce itself loop condition in the form of: "Something" is equal/unequal "something" (unequal indicated <>, more or equal symbol >=) 

Window1.Tekst1.Tekst = "introduced the text you want"
 Window1.Window1.Shirina&gt; "200"
 Window1.Button1.Fore color &lt;&gt; black
 Window1.Window1.X&gt; = Window1.Tekst1.Tekst
Then click on the button Create will block cycle tree action .

Now all action inside this loop block will always run in cycles , as long as the true condition of the cycle. Otherwise, the loop condition has also worked as usual condition .

Remark

conditions for cycles can be not only simple (a comparison of the text, numbers) and complex (mathematical formula, the comparison with the value of the properties of another object). To create complex actions, use Master complex actions (the button with the wand on the right)

Exit from the cycle can be early, not waiting until then follow the remaining steps, more on this in Useful sites . Interrupts
