Description

Support is the word marks of different values (colors, the main folders, cursors). They are used when necessary to specify the value property.

example, to make the red box is sufficient to assign property color auxiliary word "Red". Or if you want to specify certain position of the text on the button, then you want to assign the property clause of the auxiliary word "Center".

Auxiliary words do not change and do not, it's just information, which is usually used to transfer it to the properties of objects. All auxiliary words are grouped by categories (folder, position the figure, cursors) and are menu , which is caused by right-clicking on any text field development environment Algoritm2 .
