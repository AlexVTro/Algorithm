Description

For dating with the development environment Algorithm 2, let's create a simple project "The calculation of weight." This program will calculate how much your optimal weight , depending on your growth . Work it will be as follows: the user enters a window, your weight and height, and on pressing the button the program displays the result.

created this project you will get all the basic skills work in the development environment Algorithm 2 . It takes no more than 30 minutes. Well Let us get down !
