Описание

События это действия пользователей (клик, движение курсора, нажатие клавиш), которые могут отловить объекты (кнопки, надписи, окна). Раздел событий позволяет назначить объектам те события, которые будут отлавливаться ими в процессе работы вашей программы.

Создать событие для объекта можно просто выбрав его из списка событий, а затем нажать кнопку Создать. Все это в находится разделе событий Среды разработки Алгоритм 2

Таким образом, если создать событие Клик объекту Кнопка, то все действия внутри этого события будут выполняться при клике на кнопке:

т.е. при клике по кнопке текст заголовка окна изменится на "Окно1".

Примечание

Событими являются не только действия пользователя (клики, ввод текста, изменение размеров) но и события происходящие внутри объектов. Например, Изменение значения ячейки памяти, Окончание проигрывания видео файла, Секундомер тикнул и т.д.
