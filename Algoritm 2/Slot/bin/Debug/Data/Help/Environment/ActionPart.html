Описание

Действия это некие команды программе (изменить текст окна, изменить цвет кнопки), которые выполняются программой и программируются вами.

Создать действие для объекта можно зайдя в Раздел действий Среды разработки Алгоритм 2:

Здесь достаточно выбрать объект, который будет выполнять ваше действие и выбрать свойство которое будет изменяться у данного объекта. Задав чуть ниже новое значение для этого свойства и нажав кнопку Создать, вы и создадите действие:

Таким образом, когда по обеъкту Кнопка произведут Клик, то все действия внутри этого события будут выполнятся. В данном случае текст заголовка окна изменится на "Окошко"

Примечание

Новые значения для свойств могут быть не только простыми (текст, число) но и сложными (математическая формула, значения свойства у другого объекта). Для создания сложных действий воспользуйтесь Мастером сложных действий (кнопка с волшебной палочкой справа)
