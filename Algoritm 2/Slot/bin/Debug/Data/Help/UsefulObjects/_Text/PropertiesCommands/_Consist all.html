Описание

Проверяет, чтобы текст состоял исключительно из символов, переданных этой команде. Если хотя бы один символ будет не такой, команда вернет "Нет".

Данные, требуемые для выполнения свойсва или команды:

_Полезные объекты._Текст._Текст состоит из (Исходный текст, Подряд символы из которых должен состоять текст)

Данные
Описание
Исходный текст Текст оригинал, который будет использоваться данным свойством или командой.
Подряд символы из которых должен состоять текст Символы из которых должен сосотоять текст.
