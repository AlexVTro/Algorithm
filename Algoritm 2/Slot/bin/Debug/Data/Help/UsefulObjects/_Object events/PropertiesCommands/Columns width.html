Описание

С помощью этого свойства можно задавать ширину любых столбцов таблицы.

Данные, требуемые для выполнения свойсва или команды:

Объекты._Объект события.Ширина столбца (Номера столбцов через запятую)

Данные
Описание
Номера столбцов через запятую Здесь требуется указать столбцы, с котороми будут производиться действия. Если стобец один, то можно укать так: "0" - первый столбец или "2" - третий. Если несколько, то: "0,1,3" - первый второй и четвертый столбец.
