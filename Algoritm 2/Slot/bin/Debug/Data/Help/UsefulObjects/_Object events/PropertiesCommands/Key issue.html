Описание

Свойство показывает, зарегистрирована ли уже программа. Т.е. был ли введен верный ключ регистрации или нет.

Данные, требуемые для выполнения свойсва или команды:

Объекты._Объект события.Ключ активации выдать (ID пользователя, Показать в окне, )

Данные
Описание
ID пользователя Тут указывается ID того пользователя, для которого генерируется Ключ регистрации.
Показать в окне Показать результат работы в специальным окне.
