Стиль кнопки
Справка -> Объекты -> _Объект события -> Свойства и команды -> Стиль кнопки

Смотрите также, Пример
Свернуть все

Описание

Задает вид отображения объекта на форме (плоский вид, объемный).

Все возможные стили кнопки описаны в Вспомогательные слова -> Стили кнопки.

Пример

Варианты изменения cтиля кнопки у кнопки

Aлгоритм2
[копировать в буфер обмена] 
Окно1.Кнопа1.Стиль кнопки = "обычный"
Окно1.Кнопа1.Стиль кнопки = Поднимающийся
Окно1.Кнопа1.Стиль кнопки = Окно1.Кнопка2.Стиль кнопки
Окно1.Кнопа1.Стиль кнопки = Окно1.Текст1.Текст
