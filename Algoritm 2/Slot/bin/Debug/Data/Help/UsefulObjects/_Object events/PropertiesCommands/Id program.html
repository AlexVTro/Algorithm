Описание

Для каждой вашей программы генерируется её уникальный ID программы. Все ключи регистрации выдаются на опреденный ID программы. Если вы его измените, то и ранее выданные ключи (на старый ID) не подойдут на новый ID. Также неизменным должен оставаться Ключ шифрования.

Примечание

Если вы используете два объекта Триал - например, один для ввода ключа активации, другой для генерации этих ключей (как в примере Примеры\Сложные проекты\Триал), то следите, чтобы Id программы и Ключ шифрования у них были одинаковые!
