Сделать скриншот
Справка -> Полезные объекты -> _Экран -> Свойства и команды -> _Сделать скриншот

Смотрите также, Пример
Свернуть все

Описание

Данная команда фотографирует все что расположено на экране монитора и сохраняет это в буфер обмена (Место, где хранится то, что вы копируете и вставляете).

Рисунок также сохраняется во временном файле, который можно назначить рисунком объекта (например фоновым рисунком окна).

Данные, требуемые для выполнения свойсва или команды:

_Полезные объекты._Экран._Сделать скриншот (Только активное окно)

Данные
Описание
Только активное окно Если передать в качестве данных слово "Да", то будет сфотографировано с рабочего стола только окно активного приложения. Если "Нет", то всё, расположенное на экране.

Пример

Варианты изменения фонового рисунка у главного окна

Aлгоритм2
[копировать в буфер обмена] 
Окно1.Окно1.Фоновой рисунок = "C:\1.bmp"
Окно1.Окно1.Фоновой рисунок = "\1.bmp"
Окно1.Окно1.Фоновой рисунок = мои документы & "\2.jpg"
Окно1.Окно1.Фоновой рисунок = Окно1.Кнопка1.Фоновой рисунок
