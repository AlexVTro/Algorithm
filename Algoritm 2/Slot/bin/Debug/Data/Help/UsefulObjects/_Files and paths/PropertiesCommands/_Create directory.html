Описание

Команда создает папку на компьютере. Причем, например, папки "С:/a/b/c" успешно создадутся, даже если до этого не существовало даже папки "С:/a".

Данные, требуемые для выполнения свойсва или команды:

_Полезные объекты._Файлы и папки._Создать папку (Путь к файлу)

Данные
Описание
Путь к файлу Тут нужно указать путь к файлу. Путь должен обозначаться в ковычках, например "C:\1.jpg" или "\2.jpg" (во втором случае файл "2.jpg" должен располагаться в папке с проектом)
