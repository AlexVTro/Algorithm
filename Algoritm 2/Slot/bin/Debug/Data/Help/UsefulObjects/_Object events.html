Описание

Объект события предоставляет доступ к свойствам и командам того объекта, у которого в данный момент происходит какое-нибудь событие.

Например, в данном случае

Объект события предоставляет доступ к свойствам и командам Кнопки1, т.к. он находится в событии Клик этой кнопки.

Таким образом, Объект события не предоставляет ничего нового, он лишь упрощает создание действий. Ведь теперь, если требуется изменить свойство объекта, над которым произошло событие, не обязательно указывать его имя (например Кнопка1) а можно просто указать универсальное имя Объект события. Это бывает очень полезно например в Массивах.

В Объект события заранее внесены все свойства, которые только возможны при разных событиях и разных объектах. Поэтому не все свойства и команды доступны в каждом отдельном событии, а только свойства и команды объекта, над которым произошло данное событие. Например в данном случае, из всех свойств и команд Объект события будут работать только те, которые есть у объекта Кнопка1 (в данном случае Текст).

Все возможные свойства и команды объекта:
Свойства и команды Свойства - это настройки объектов (цвет, текст, размер), изменяя которые вы можете настроить объект необходимым образом. Объекты также могут выполнять некоторые действия над собой, которые называются командами.
