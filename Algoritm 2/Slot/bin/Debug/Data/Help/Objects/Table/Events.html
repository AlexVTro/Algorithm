Описание

События это действия пользователей (клик, движение курсора, нажатие клавиш), которые могут отловить объекты.

У каждого объекта есть определенные события, которые он может отлавливать. Устанавлевается отлов событий для объектов в Разделе событий. После этого все действия, которые вы расположите внутри этого события в Разделе дерева программы, будут выполняться после возникновения данного события (например клика).

Вращение колесикаСобытие происходит у активного сейчас объекта (объекта в фокусе), в момент, когда пользователь начинает вращать колесико мыши. У объекта, на котором произошло данное событие, появляются дополнительные свойства.
Движение курсораСобытие происходит у объекта, когда пользователь двигает по нему мышкой. У объекта, на котором произошло данное событие, появляются дополнительные свойства.

Наведение курсораСобытие вызывается как только курсор попадает на объект.
Нажатие вниз кнопки Событие происходит у активного сейчас объекта (объекта в фокусе), в момент, когда пользователь надавливает на кнопку клавиатуры. У объекта, на котором произошло данное событие, появляются дополнительные свойства.
Нажатие клавиатуры Событие происходит у активного сейчас объекта (объекта в фокусе), в момент, когда пользователь нажимает кнопку клавиатуры. У объекта, на котором произошло данное событие, появляются дополнительные свойства.
Нажатие кнопки мыши Событие происходит у активного сейчас объекта (объекта в фокусе), в момент, когда пользователь надавливает кнопку мыши. У объекта, на котором произошло данное событие, появляются дополнительные свойства.

Отведение курсораСобытие вызывается как только курсор покидает на объект.
Отжатие кнопки Событие происходит у активного сейчас объекта (объекта в фокусе), в момент, когда пользователь отпускает, нажатую ранее кнопку клавиатуры. У объекта, на котором произошло данное событие, появляются дополнительные свойства.
Отжатие кнопки мыши Событие происходит у объекта, когда пользователь отпускает, нажатую ранее кнопку мыши. У объекта, на котором произошло данное событие, появляются дополнительные свойства.
Получение фокуса Событие происходит, когда объекту передали фокус (сделали объект активным)
Потеря фокуса Событие происходит, когда у активного объекта, забрали фокус (другой объект стал активным)
ПрокруткаСобытие происходит, когда у объекта двигается какая-либо его полоса прокрутки. Обычно, чтобы сделать возможность прокрутки объекта, нужно установить свойство "Прокрутка" равным "Да".
Прорисовка Событие происходит, когда объект прорисовывается полностью или частично. Это происходит если он был скрыт за каким-нибудь другим объектом, а потом стал виден полностью. Или по нему пробежал курсор мыши и объект перерисовывается, чтобы стереть с себя курсор.
Создание Событие происходит в самом начале работы программы. В момент, когда объект впервые создается.
