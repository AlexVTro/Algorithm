Описание

Возвращает номер строки, которая будет следующей в области выборки. Если, например, до этого запрашивали первую строку, то сейчас команда выдаст 2-ю. В следующий раз 3-ю и т.д.

Данные, требуемые для выполнения свойсва или команды:

Объекты.Таблица.Номер следующей строки (Номер начальной строки, Область выборки)

Данные
Описание
Номер начальной строки Строка, с которой начинают выборку. Первая строка имеет номер "0".
Область выборки Указывает где искать строку. Например если область выбоки "Выделенные строки" то данная функция будет видеть только выделенные строки и искать только в них.
