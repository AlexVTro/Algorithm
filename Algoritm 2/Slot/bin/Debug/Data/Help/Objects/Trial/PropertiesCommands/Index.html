Описание

Наряду с именем, номер является опозновательным свойством объекта. По этому свойству можно отличать объекты, например, если у них одинаковые имена (в массивах)

Номер может быть любым положительным числом. Кроме того, когда вы создаете действия и требуется указать номер объекта, то можно использовать вспомогательное слово Все. В этом случае, созданное дейсвтие будет распространяться на все объекты с указанным вами именем, вне зависимости от номера.

Примечание

У двух объектов не может быть одинаковых имен и одновременно номеров. Эти два свойства созданы для однозначной идиентификации объекта в среде разработки.
