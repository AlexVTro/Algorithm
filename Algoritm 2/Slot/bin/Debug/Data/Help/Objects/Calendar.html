Описание

Календарь предоставляет возможность пользователю удобно выбрать дату.

Календарь имеет удобный интерфейс для выбора даты. Дату, которую сейчас выбрали, можно узнать из свойства Выбранная дата

Возможности этого объекта:
События События это действия пользователей (клик, движение курсора, нажатие клавиш), которые могут отловить объекты.
Свойства и команды Свойства - это настройки объектов (цвет, текст, размер), изменяя которые вы можете настроить объект необходимым образом. Объекты также могут выполнять некоторые действия над собой, которые называются командами.
