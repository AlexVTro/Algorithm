Описание

Основной цвет, в который окрашен объект.

Все цвета могут задаваться словесным обозначением цвета (красный, синий, черный), подробнее об этом в разделе Вспомогательные слова -> Цвета. Либо цвет можно указать в специальной форме, где сначала идет интенсивность красного цвета, потом через точку с запятой интенсивность зеленого, а еще через точку с запятой синего. Максимальная интенсивность 255, минимальная 0. Например: 

"255; 0; 0;" = красный
"0; 0; 255;" = синий
"0; 0; 0;" = черный
"255; 255; 255;" = белый
"255; 255; 128;" = светло желтый

Примечание

Если вы хотите сделать цвет прозрачным (не все объекты поддерживают эту возможность) присвойте этому свойству значение "Никакой" или "".

Пример

Варианты изменения цвета у главного окна

Aлгоритм2
[копировать в буфер обмена] 
Окно1.Окно1.Цвет = красный
Окно1.Окно1.Цвет = "255; 201; 128;"
Окно1.Окно1.Цвет = "128;128;0;"
Окно1.Окно1.Цвет = Окно1.Кнопка1.Цвет шрифта
