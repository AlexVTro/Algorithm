Привязка
Справка -> Объекты -> Панель -> Свойства и команды -> Привязка

Смотрите также, Пример
Свернуть все

Описание

Привязка помогает правильно располагать объекты на окне, при изменении размеров окна пользователем.

Можно привязать объект к какому либо краю окна и при изменении размеров окна, объект будет следовать за краем, к которому привязан. Все возможные привязки описаны в Вспомогательные слова -> Привязки.

Пример

Варианты изменения привязки у кнопки

Aлгоритм2
[копировать в буфер обмена] 
Окно1.Кнопа1.Привязка = "Слева"
Окно1.Кнопа1.Привязка = справа_сверху_снизу
Окно1.Кнопа1.Привязка = Окно1.Кнопка2.Привязка
Окно1.Кнопа1.Привязка = Окно1.Текст1.Текст
