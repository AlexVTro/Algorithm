Обновить версию:
1. В проекте в файле Peremens.vb изменить Version = "x.x"
2. ПереСкомпилируй проект на всякий случай в Release
3. Запустить AlgBuild (из визуал студии чтобы не упал) и нажми Build&Publish
4. algoritm2.ru\api файлы LastVersionEn.php и LastVersionRu.php вставить новую версию и загрузить на сервер
5. Зайти в админку алга http://algoritm2.ru/administrator/index.php?option=com_languages&view=overrides логин admin_alg пароль 7.. и изменить констатну ALG_VERSION для обоих языков
