Описание

Позволяет вставить запись в список в определенное место по номеру записи

Этим свойством можно вставить в спискок конкретную нужную нам запись в то место, куда мы хотим чтобы она поместилась. Место записи выбирается по номерам записей списка. Например: номер первой записи - 0, второй - 1, третьей - 2 и т.д. Чтобы вставить, например новую запись третьей по списку, в поле "Место для записи списка" ставим значение "4".

Данные, требуемые для выполнения свойсва или комманды:

Объекты.Большой список.Вставить запись (Место для записи в списке, Текст записи)

Данные
Описание
Место для записи в списке Порядковый номер записи, куда вы хотите вставить запись. Первая запись имеет номер "0", вторая "1" и т.д.
Текст записи Текст записи в списке.

Примечание

Если номер вставляемой записи в списке уже существует, новая запись заменит ее, а старую поставит на номер выше (визуально ниже) по списку.
