Описание

С помощью этого объекта можно сделать из вашей программы полноценную ShareWare. Он возьмет на себя все заботы об отслеживании триального периода, об активации, о ключе регистрации и т.д.

Триал сделает регистрацию программ автоматической. Пример использования этого объекта есть в папке Примеры\Сложные проекты\Триал

Объект не дает никакой гарантии, что вашу ShareWare программу не взломают. Хакеру это не составит особо большого труда. Поэтому, советуем совмещать объект Триал, со своими методами слежения триального периода и активации. Дабы усложнить работу взломщику.

Возможности этого объекта:
События События это действия пользователей (клик, движение курсора, нажатие клавиш), которые могут отловить объекты.
Свойства и комманды Свойства - это настройки объектов (цвет, текст, размер), изменяя которые вы можете настроить объект необходимым образом. Объекты также могут выполнять некоторые действия над собой, которые называются коммандами. Изменять свойства можно вручную в Разделе свойств. Или запрограммировать их изменение в Разделе действий. Во втором случае сначала для объекта создается событие в Разделе событий, при котором данное свойство должно поменяться. Примеры использования свойств и комманд в Разделе действий:

Примечание

Каждый объект Триал ассоциируется с вашей программой только по свойству Id программы (уникальный идентификатор только вашей программы) и по свойству Ключ шифрования (ключ, указывающий, алгоритм шифрования). Поэтому, если вы используете несколько объектов Триал и хотите чтобы они работали с одними ключами и Id, то следите, чтобы Id программы и Ключ шифрования у них были одинаковые!
