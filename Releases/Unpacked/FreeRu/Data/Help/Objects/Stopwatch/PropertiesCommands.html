Описание

Свойства - это настройки объектов (цвет, текст, размер), изменяя которые вы можете настроить объект необходимым образом. Объекты также могут выполнять некоторые действия над собой, которые называются коммандами.

Изменять свойства можно вручную в Разделе свойств. Или запрограммировать их изменение в Разделе действий. Во втором случае сначала для объекта создается событие в Разделе событий, при котором данное свойство должно поменяться. Примеры использования свойств и комманд в Разделе действий:

Окно1.Объект1.Текст = "Новый текст1"
Окно1.Объект2.Рисунок = "C:\Картинка.jpg"
Окно1.Объект2.Получить фокус

Список свойств объекта Секундомер:
XКоордината X левого верхнего угла объекта.
YКоордината Y левого верхнего угла объекта.
Вспомогательное полеДанное свойство никак не влияет на объект. В него вы можете записать что угодно, оно создано исключительно для использования вами.
ИмяКо всем объектам в Алгоритме 2 обращаются по именам. По этому свойству можно однозначно указать объект, который вы хотите использовать.
Интервал отсчетаОпределяет, через какое количество времени(в миллисекундах) повторить определенные действия.
НомерНаряду с именем, номер является опознавательным свойством объекта. По этому свойству можно отличать объекты, например, если у них одинаковые имена (в массивах)
Прошло интерваловВозвращает количество пройденных интервалов секундомером. Если в свойстве "Интервал отсчета" было введено значение 1000 (т.е. 1 секунда), после запуска секундомера и истечения например 5 секунд, свойство Прошло интервалов вернет значение 5 (т.е. пошло 5 интервалов по 1ой секунде).
Работает Объекты в любой момент можно сделать нерабочим. Он будет присутствовать на окне, но пользователь ничего с ним не сможет сделать (нажать, изменить текст, размер)
СтартДанное свойство запускает отсчет секундомера.
СтопДанное свойство останавливает отсчет секундомера.
Тип Это свойство нельзя изменить, оно служит лишь для того что бы сообщить вам какого типа объект (текстовое поле, кнопка, окно).
