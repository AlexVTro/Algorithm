Описание

Свойство позволяет узнать содержимое любой ячейки таблицы, либо изменить его.

Данные, требуемые для выполнения свойсва или комманды:

Объекты.Таблица.Значение ячейки (Номера строк через запятую, Номера столбцов через запятую)

Данные
Описание
Номера строк через запятую Перечислите нужные строки через запятую в двойных кавычках, например, "0,1,8,3" или "2".
Номера столбцов через запятую Здесь требуется указать столбцы, с которыми будут производиться действия. Если столбец один, то можно укать так: "0" - первый столбец или "2" - третий. Если несколько, то: "0,1,3" - первый второй и четвертый столбец.
