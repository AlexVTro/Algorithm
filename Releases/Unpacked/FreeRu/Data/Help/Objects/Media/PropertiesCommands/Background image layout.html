Стиль фона
Справка -> Объекты -> Медиа -> Свойства и комманды -> Стиль фона

Смотрите также, Пример
Свернуть все

Описание

Стиль отображения фонового рисунка на объекте (замостить плиткой, растянуть на объект).

Все возможные стили фона описаны в Вспомогательные слова -> Стили фона.

Пример

Варианты изменения cтиля фона у кнопки

Aлгоритм2
[копировать в буфер обмена] 
Окно1.Кнопа1.Стиль фона = "Масштабируемый"
Окно1.Кнопа1.Стиль фона = Растянутый
Окно1.Кнопа1.Стиль фона = Окно1.Кнопка2.Стиль фона
Окно1.Кнопа1.Стиль фона = Окно1.Текст1.Текст
