Открыть медиафайл
Справка -> Объекты -> Медиа -> Свойства и комманды -> Открыть медиафайл

Смотрите также, Пример
Свернуть все

Описание

Данным свойством выбираем, какой файл хотим проиграть. Только после того, как открыли медиафайл, его можно проиграть свойством "Плей".

Данные, требуемые для выполнения свойсва или комманды:

Объекты.Медиа.Открыть медиафайл (Путь к файлу)

Данные
Описание
Путь к файлу Тут нужно указать путь к файлу. Путь должен обозначаться в кавычках, например "C:\1.jpg" или "\2.jpg" (во втором случае файл "2.jpg" должен располагаться в папке с проектом)

Примечание

Пример

Пример открытия mp3 файла и его последующее проигрывание:

Aлгоритм2
[копировать в буфер обмена] 
Окно1.Аудио1.Открыть медиафайл = "С:\Track1.mp3"
Окно1.Аудио1.Плей
