Описание

Позволяет взять из текста любую его часть, если известен порядковый номер первого символа части и ее длинна. Например, из текста "Привет мир!" можно получить второе слово ("мир") указав, что начало куска 8-ой символ, а длинна куска 3 символа. В этом случае результатом работы комманды будет слово "мир".

Данные, требуемые для выполнения свойсва или комманды:

_Полезные объекты._Текст._Взять кусок текста (Исходный текст, Номер символа начала куска, Длинна куска)

Данные
Описание
Исходный текст Текст оригинал, который будет использоваться данным свойством или коммандой.
Номер символа начала куска Номер символа, с которого надо начинать брать кусок текста. Первый символ имеет номер "1", второй "2" и т.д.
Длинна куска Количество символов какой длинны будет кусок текста.
