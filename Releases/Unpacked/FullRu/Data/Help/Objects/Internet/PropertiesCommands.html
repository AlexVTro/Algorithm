Описание

Свойства - это настройки объектов (цвет, текст, размер), изменяя которые вы можете настроить объект необходимым образом. Объекты также могут выполнять некоторые действия над собой, которые называются коммандами.

Изменять свойства можно вручную в Разделе свойств. Или запрограммировать их изменение в Разделе действий. Во втором случае сначала для объекта создается событие в Разделе событий, при котором данное свойство должно поменяться. Примеры использования свойств и комманд в Разделе действий:

Окно1.Объект1.Текст = "Новый текст1"
Окно1.Объект2.Рисунок = "C:\Картинка.jpg"
Окно1.Объект2.Получить фокус

Список свойств объекта Интернет:
_Ip источникаВозвращает IP-адрес (или ссылку на сайт), с которым, в данный момент, происходит передача данных.
_Всего байтДанное свойство возвращает полный размер всего загружаемого файла в байтах.
_Отменить событиеСвойство позволяет отменить произошедшее событие, т.е. сделать так, как-будто события и не было. Делается это изменением значения свойства на "Да".
_Получено байтДанное свойство возвращает количество загруженных данных (в байтах.)
_Процент полученногоВозвращает процент загруженного файла. Мин. значение 0, макс. 100
XКоордината X левого верхнего угла объекта.
YКоордината Y левого верхнего угла объекта.
Автоматически перенаправлятьсяСвойство определяет, будет ли произведено автоматическое перенаправление при получении ссылки перенаправления или нет.
В фокусеЕсли в фокусе равняется "Да", то объект является активным и именно с ним сейчас работает пользователь (например вводит текст в текстовое поле).
ВидимыйОбъекты в любой момент можно сделать видимыми или невидимыми изменяя это свойство.
Время задержкиПозволяет задать время задержки между запросами измеряемое в миллисекундах (мс.)
Вспомогательное полеДанное свойство никак не влияет на объект. В него вы можете записать что угодно, оно создано исключительно для использования вами.
Выполнить запросКоманда выполняет ранее заданный запрос
ВышинаРазмер объекта по вертикали, измеряемый в пикселях.
Длинна содержимогоХедер (Header) Content-Lenght. Возвращает длинну содержимого GET запроса.
Заголовки запросаСвойство возвращает так называемые Headers(Заголовки) запросов.
ИмяКо всем объектам в Алгоритме 2 обращаются по именам. По этому свойству можно однозначно указать объект, который вы хотите использовать.
Код результатаВозвращает код результата запроса к серверу.
Кодировка страницыВозвращает, устанавливает кодировку страницы (utf-8, windows-1251 и т.д.)
Куки запросовПри авторизации на сайте выдаются автоматические куки запросов, которые и записываются в это свойство.
Максимальная вышинаМаксимально разрешенная вышина объекта. Если не нужно ограничивать по высоте объект, присвойте этому свойству значение "0"
Максимальная ширинаМаксимально разрешенная ширина объекта. Если не нужно ограничивать по ширине объект, присвойте этому свойству значение "0"
Метод запросаОпределяет каким методом будет произведен http запрос. Принимаемые значения POST/GET. POST – для отправки данных, GET – для получения.
Минимальная вышинаМинимально разрешенная вышина объекта. Если не нужно ограничивать по высоте объект, присвойте этому свойству значение "0"
Минимальная ширинаМинимально разрешенная ширина объекта. Если не нужно ограничивать по ширине объект, присвойте этому свойству значение "0"
Наличие соединенияПроверяет наличие соединения с сервером.
НомерНаряду с именем, номер является опознавательным свойством объекта. По этому свойству можно отличать объекты, например, если у них одинаковые имена (в массивах)
Обновитькомманда заново прорисовывает объект на окне.
Очистить кукиКоманда удаляет полученные ранее куки сайта.
Папка для загрузокОпределяет в какое место будет загружен файл
Перенести назадкомманда переносит объект на задний план. Объект становится ниже всех прочих объектов.
Перенести напередкомманда переносит объект на передний план. Объект становится выше всех прочих объектов.
ПодсказкаПри наведении курсора на объект, всплывает подсказка которую вы введете в ее поле
Получить код страницыКоманда Возвращает html код той страницы, которая была введена Вами в поле "Ссылка" свойства "Получить код страницы"
Получить фокускомманда получить фокус делает объект активным и именно с ним будет работает пользователь (например получить фокус текстовым полем, и последующие нажатия кнопок пользователя будут печататься в текстовом поле).
ПривязкаПривязка помогает правильно располагать объекты на окне, при изменении размеров окна пользователем.
ПриниматьУказывает какой тип данных принимать программе.
Прокси IP
Прокси порт
Работает Объекты в любой момент можно сделать нерабочим. Он будет присутствовать на окне, но пользователь ничего с ним не сможет сделать (нажать, изменить текст, размер)
Размер буфераДанное свойство определяет какими фрагментами (по сколько байт) будет скачиваться файл. Дефолтное значение 50 000 байт
РастяжкаРастягивает объект вдоль какой-нибудь границы окна (левой, нижней). И эта растяжка сохраняется (продолжает растягиваться) при изменении пользователем размеров окна.
Результат запросаВозвращает результат выполненного запроса. Напрм., если Вы выполняли GET запрос страницы, то в этом свойстве будет находиться html текст запрошенной сраницы
Скачать файлСкачивает файл с сети Интернет по ссылке которой Вы указали в поле "Ссылка", свойства "Скачать файл".
Скачивается файлСвойство определяет скачивается ли файл в данный момент, и возвращает сооттветсвенно значения "Да" или "Нет"
Скачка возобновитьКак видно из названия, действие возобновляет скачку файла, если она была приостановлена ранее.
Скачка отменитьОбрывает скачку файла.
Скачка паузаПриостанваливает скачку файла, которую можно восстановить свойством Скачка возобновить
Содержимое запросаСюда подставляются различные аргументы запроса.
Ссылка запросаУстанавливает ссылку для запроса (напрм. http://algoritm2.ru)
Ссылка откуда пришлиТак называемый Реферер (referer). Указывает серверу ссылку с которой пришел пользователь. Обычно устанавливается начальная страница (напрм. http://algoritm2.ru). Без него сервер может ответить 403 (Access Forbidden), если не допускаются запросы с других сайтов.
Ссылка перенаправленияСвойство возвращает ссылку перенаправления, если такова была в отвеченном сервером сообщении.
Стиль рамкиУстанавлвиает стиль рамки объекта.
Стиль фонаСтиль отображения фонового рисунка на объекте (замостить плиткой, растянуть на объект).
ТабНомерПри нажатии кнопки Tab на клавиатуре, все объекты окна по очереди активизируются (становятся в фокусе). Чем меньше Таб номер, тем дальше в очереди на активизацию объект.
ТабСтопПри нажатии кнопки Tab на клавиатуре, все объекты окна по очереди активизируются (становятся в фокусе). Данное свойство управляет тем, надо ли активизировать этот объект когда подошла его очередь, или пропустить его активизацию.
ТаймаутУстанавливает время в миллисекундах. Если через это время не будет ответа, тогда сервер не отвечает.
Тип Это свойство нельзя изменить, оно служит лишь для того что бы сообщить вам какого типа объект (текстовое поле, кнопка, окно).
Тип браузераДанное свойство позволяет представляться серверу каким-либо браузером (напрм. Mozilla/4.0, Opera/10.0)
Тип содержимогоВозвращает тип полученного содержимого(текст, медафайл, текстовой файл и т.д.) в определенном формате.
Удерживать соединениеУдерживает соединение с сервером или нет
Фоновой рисунокРисунок, который будет расположен на заднем фоне объекта.
ЦветОсновной цвет, в который окрашен объект.
ШиринаРазмер объекта по горизонтали, измеряемый в пикселях.
Язык страницыСервер возвращает язык запрошенной страницы.
