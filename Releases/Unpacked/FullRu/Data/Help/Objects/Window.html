Описание

Это главный объект. На окнах располагаются все остальные объекты.

Окно это такой же обычный объект программы, для которого можно создавать события и свойства. Его главное предназначение, это расположение на нем остальных объектов. Окон в программе может быть любое количество. Которое из окон будет главным определяет свойство Главное окно.

Возможности этого объекта:
События События это действия пользователей (клик, движение курсора, нажатие клавиш), которые могут отловить объекты.
Свойства и комманды Свойства - это настройки объектов (цвет, текст, размер), изменяя которые вы можете настроить объект необходимым образом. Объекты также могут выполнять некоторые действия над собой, которые называются коммандами. Изменять свойства можно вручную в Разделе свойств. Или запрограммировать их изменение в Разделе действий.
