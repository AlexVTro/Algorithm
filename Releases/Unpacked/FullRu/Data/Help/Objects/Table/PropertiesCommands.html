Описание

Свойства - это настройки объектов (цвет, текст, размер), изменяя которые вы можете настроить объект необходимым образом. Объекты также могут выполнять некоторые действия над собой, которые называются коммандами.

Изменять свойства можно вручную в Разделе свойств. Или запрограммировать их изменение в Разделе действий. Во втором случае сначала для объекта создается событие в Разделе событий, при котором данное свойство должно поменяться. Примеры использования свойств и комманд в Разделе действий:

Окно1.Объект1.Текст = "Новый текст1"
Окно1.Объект2.Рисунок = "C:\Картинка.jpg"
Окно1.Объект2.Получить фокус

Список свойств объекта Таблица:
_Введенный символ Буквенный символ (а, б, 3), который ввели с клавиатуры в данное событие.
_Кнопка мышиХранит название кнопки мыши (левая кнопка, колесико), которой произвели событие.
_Количество кликов Содержит число кликов, которые произвели в данное событие.
_Количество строк
_Мышь X Хранит координату X курсора в данный момент времени. Свойство доступно, например, в событии "Нажатие кнопки мыши", если выбрать в списке объектов тот объект, для которого и создано это событие.
_Мышь Y Хранит координату Y курсора в данный момент времени. Свойство доступно, например, в событии "Нажатие кнопки мыши", если выбрать в списке объектов тот объект, для которого и создано это событие.
_Нажат альт Отображает, нажата ли клавиша клавиатуры альт (Alt) в данный момент или нет.
_Нажат контрол Отображает, нажата ли клавиша клавиатуры контрол (Ctrl) в данный момент или нет.
_Нажат шифт Отображает, нажата ли клавиша клавиатуры шифт (Shift) в данный момент или нет.
_Нажатая клавиша Содержит наименование клавиши клавиатуры (а, б, ctrl, enter), которую нажали во время события.
_Номер начальной строкиВозвращает номер строки, которая ближе к началу, чем возможные другие строки, задействованные в сработавшем событии.
_Номер столбцаВозвращает номер столбца.
_Номер строкиВозвращает номер строки.
_Отменить вводДанное свойство позволяет отменить введенный пользователем символ. Делается это изменением значения свойства на "Да".
_Отменить событиеСвойство позволяет отменить произошедшее событие, т.е. сделать так, как-будто события и не было. Делается это изменением значения свойства на "Да".
_Прокручено колесиком Содержит количество раз, сколько повернули колесико, в данное событие.

Номер первой строки Возвращает номер строки, которая будет первой в области выборки.
Номер последней строки Возвращает номер строки, которая будет последней в области выборки.
Номер предыдущей строки Возвращает номер строки, которая располагается перед текущей записью в области выборки. Если, например, до этого запрашивали последнюю строку, то сейчас комманда выдаст пред-последнюю. В следующий раз пред-пред-последнюю и т.д.
Номер следующей строки Возвращает номер строки, которая будет следующей в области выборки. Если, например, до этого запрашивали первую строку, то сейчас комманда выдаст 2-ю. В следующий раз 3-ю и т.д.
Номер столбца по координатам Возвращает номер столбца, на который попадают переданные координаты X и Y. Координаты можно взять, например, из свойств _Мышь X, _Мышь Y.
Номер строки по координатам Возвращает номер строки, на которую попадают переданные координаты X и Y. Координаты можно взять, например, из свойств _Мышь X, _Мышь Y.
Номера выделенных столбцов Свойство выделяет столбцы таблицы или показывает, какие столбцы выделены. Результат выдается, например, в такой форме: 0,2,3. Т.е. выделены первый, третий и четвертый столбец (т.к. нумерация столбцов таблицы начинается с 0; 0 - первый столбец, 1 - второй и т.д.)
Номера выделенных строк Свойство выделяет строки таблицы или показывает, какие строки выделены. Результат выдается, например, в такой форме: 0,2,3. Т.е. выделена первая, третья и четвертая строка (т.к. нумерация строк таблицы начинается с 0; 0 - первая строка, 1 - вторая и т.д.)

Также знайте, что открывается только Excel 8.0. Если ваша таблица Excel не открывается программой, то используйте эту: Проекты\Примеры\Сложные проекты\Таблица2\База Excel.xls

Полосы прокрутки
Получить фокускомманда получить фокус делает объект активным и именно с ним будет работает пользователь (например получить фокус текстовым полем, и последующие нажатия кнопок пользователя будут печататься в текстовом поле).
ПривязкаПривязка помогает правильно располагать объекты на окне, при изменении размеров окна пользователем.
Работает Объекты в любой момент можно сделать нерабочим. Он будет присутствовать на окне, но пользователь ничего с ним не сможет сделать (нажать, изменить текст, размер)
РастяжкаРастягивает объект вдоль какой-нибудь границы окна (левой, нижней). И эта растяжка сохраняется (продолжает растягиваться) при изменении пользователем размеров окна.
Режим выделения Режим выделения указывает, как будут выделяться ячейки когда по ним будет кликать пользователь - по одной ячейке или вся строка целиком.
Режим редактирования Режим редактирования указывает, может ли пользователь редактировать таблицу просто кликая по ней и вводя данные.
Сортировать комманда сортирует таблицу по указанному столбцу.
Сохранить Access комманда сохраняет таблицу в базе данных формата Access. При этом нельзя сохранять в новом файле - база данных Access обязательно должна уже существовать и быть открыта уже в этой таблице коммандой Открыть Access.
Сохранить таблицу комманда сохраняет таблицу, в указанном вами файле. Сохранение идет в простой текстовой форме, поэтому сохраненный файл можно легко открывать любым способом (например, Блокнотом) и редактировать.

Столбец только для чтения С помощью этого свойства можно запретить редактирование любых столбцов таблицы.
Столбцы С помощью этого свойства можно указать количество столбцов и их заголовки. Задаются столбцы в форме: "Столбец1", "Столбец2". Где Столбец1 - заголовок первого столбца, Столбец2 - заголовок второго.
Строка только для чтения С помощью этого свойства можно запретить редактирование любых строк таблицы.
Строки Строки - это все данные, записанные в таблице. Задаются строки в такой форме: "Строка1" | "Строка1", "Строка2" | "Строка2". Строки разделяются запятыми (","). Т.о. первая строка данных таблицы в этой записи - "Строка1" | "Строка1", вторая строка - "Строка2" | "Строка2". Ячейки внутри строки разделяются символом ("|"). Т.е. строка "Строка1" | "Строка1" состоит из двух ячеек, в каждой из которых записано Строка1.
ТабНомерПри нажатии кнопки Tab на клавиатуре, все объекты окна по очереди активизируются (становятся в фокусе). Чем меньше Таб номер, тем дальше в очереди на активизацию объект.
ТабСтопПри нажатии кнопки Tab на клавиатуре, все объекты окна по очереди активизируются (становятся в фокусе). Данное свойство управляет тем, надо ли активизировать этот объект когда подошла его очередь, или пропустить его активизацию.
