Описание

Свойство выделяет строки таблицы или показывает, какие строки выделены. Результат выдается, например, в такой форме: 0,2,3. Т.е. выделена первая, третья и четвертая строка (т.к. нумерация строк таблицы начинается с 0; 0 - первая строка, 1 - вторая и т.д.)

Чтобы выделить, например, блок ячеек следует комбинировать эту комманду с коммандой Номера выделенных столбцов: 

Окно1.Таблица1.Номера выделенных строк = "1,0"
Окно1.Таблица1.Номера выделенных столбцов = "1"
Данный фрагмент кода выделяет первую и вторую ячейку во втором столбце.
