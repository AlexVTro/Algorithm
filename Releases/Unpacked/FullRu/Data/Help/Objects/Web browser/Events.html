Описание

События это действия пользователей (клик, движение курсора, нажатие клавиш), которые могут отловить объекты.

У каждого объекта есть определенные события, которые он может отлавливать. Устанавливается отлов событий для объектов в Разделе событий. После этого все действия, которые вы расположите внутри этого события в Разделе дерева программы, будут выполняться после возникновения данного события (например клика).

Вперед можно изменилосьВозможность перейти Вперед в объекте Браузер может присутствовать, а может и отсутствовать. В момент изменения этой возможности и срабатывает данное событие.
Изменилась видимость
Изменились размерыСобытие происходит когда у объекта изменились размеры (ширина, вышина).
Назад можно изменилосьВозможность перейти Назад в объекте Браузер может присутствовать, а может и отсутствовать. В момент изменения этой возможности и срабатывает данное событие.
Открытие в новом окне Событие происходит, когда страница хочет открыться в новом окне. У объекта, на котором произошло данное событие, появляются дополнительные свойства.
Переходит по ссылкеСобытие срабатывает перед тем, как произойдет переход на новую страницу. У объекта, на котором произошло данное событие, появляются дополнительные свойства.
Перешел по ссылкеСобытие срабатывает, когда пользователь совершил переход по ссылке и страница загрузилась. У объекта, на котором произошло данное событие, появляются дополнительные свойства.
Получение фокуса Событие происходит, когда объекту передали фокус (сделали объект активным)
Потеря фокуса Событие происходит, когда у активного объекта, забрали фокус (другой объект стал активным)
Статусная строка измениласьСтатусная строка изменяется постоянно, в зависимости от того, что сейчас происходит в браузере. В каждый момент изменения срабатывает данное событие.
Страница загружаетсяСобытие постоянно вызывается пока интернет-страница находится в стадии загрузки в Браузер.
Страница загрузиласьСобытие вызывается когда интернет страница полностью загрузится и отобразится Браузером. У объекта, на котором произошло данное событие, появляются дополнительные свойства.
