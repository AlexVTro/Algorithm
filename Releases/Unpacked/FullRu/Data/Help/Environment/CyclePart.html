Описание

Циклы это теже самые условия, за исключением того, что все действия внутри циклов будут выполняться постоянно, пока условие остается верным (например, пока вышина окна больше 100)

Создать цикл можно зайдя в Раздел циклов Среды разработки Алгоритм 2:

Здесь следует ввести само условие цикла в виде: "Что-то" равно/неравно "чему-то" (неравно обозначается значком <>, больше либо равно значком >=) 

Окно1.Текст1.Текст = "Ввели нужный текст"
Окно1.Окно1.Ширина > "200"
Онко1.Кнопка1.Цвет шрифта <> черный
Окно1.Окно1.X >= Окно1.Текст1.Текст
Затем клик по кнопке Создать создаст блок цикла в дереве действий.

Теперь все действия внутри этого блока цикла будут постоянно циклично выполняться, пока будет верно условие цикла. В остальном, условие цикла работает также как обычное условие.

Примечание

Условия для циклов могут быть не только простыми (сравнение текста, чисел) но и сложными (математическая формула, сравнение со значением свойства другого объекта). Для создания сложных действий воспользуйтесь Мастером сложных действий (кнопка с волшебной палочкой справа)

Выйти из цикла можно досрочно, не ожидая пока выполнятся оставшиеся действия, подробнее об этом в Полезные объекты.Прерывания
