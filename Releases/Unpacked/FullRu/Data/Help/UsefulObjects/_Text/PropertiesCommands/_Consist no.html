Описание

комманда проверяет наличие любых символов в исходном тексте. Если хотя бы один из указанных символов присутствует в тексте, то комманда вернет значение "Нет", если ни один из перечисленных символов не содержится в тексте, то комманда будет равна значению "Да". Например, если в тексте "ывАаы" искать символы "па8", то комманда возвратит "Нет" (т.е. текст содержит символ "а").

Данные, требуемые для выполнения свойсва или комманды:

_Полезные объекты._Текст._Текст не содержит (Исходный текст, Подряд символы которые не должен содержать текст)
