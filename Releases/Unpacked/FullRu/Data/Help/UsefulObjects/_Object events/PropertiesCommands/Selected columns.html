Описание

Свойство выделяет столбцы таблицы или показывает, какие столбцы выделены. Результат выдается, например, в такой форме: 0,2,3. Т.е. выделены первый, третий и четвертый столбец (т.к. нумерация столбцов таблицы начинается с 0; 0 - первый столбец, 1 - второй и т.д.)

Чтобы выделить, например, блок ячеек следует комбинировать эту комманду с коммандой Номера выделенных строк: 

Окно1.Таблица1.Номера выделенных строк = "1,0"
Окно1.Таблица1.Номера выделенных столбцов = "1"
Данный фрагмент кода выделяет первую и вторую ячейку во втором столбце.
