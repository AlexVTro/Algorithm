Описание

События это действия пользователей (клик, движение курсора, нажатие клавиш), которые могут отловить объекты.

У каждого объекта есть определенные события, которые он может отлавливать. Устанавливается отлов событий для объектов в Разделе событий. После этого все действия, которые вы расположите внутри этого события в Разделе дерева программы, будут выполняться после возникновения данного события (например клика).
