Описание

Свойства - это настройки объектов (цвет, текст, размер), изменяя которые вы можете настроить объект необходимым образом. Объекты также могут выполнять некоторые действия над собой, которые называются коммандами.

Изменять свойства можно вручную в Разделе свойств. Или запрограммировать их изменение в Разделе действий. Во втором случае сначала для объекта создается событие в Разделе событий, при котором данное свойство должно поменяться. Примеры использования свойств и комманд в Разделе действий:

Окно1.Объект1.Текст = "Новый текст1"
Окно1.Объект2.Рисунок = "C:\Картинка.jpg"
Окно1.Объект2.Получить фокус

Список свойств объекта _Файлы и папки:
_Архивный Данное свойство показывает, является ли указанный в скобках файл архивным или нет.
_Временный Данное свойство показывает, является ли указанный в скобках файл временным или нет.
_Время доступа Данное свойство показывает, либо изменяет время, когда к файлу последний раз обращались. Например, "16.07.2009 0:29:31".
_Время изменения Данное свойство показывает, либо изменяет время, когда файл последний раз редактировали. Например, "16.07.2009 0:29:31".
_Время создания Данное свойство показывает, либо изменяет время, когда был создан файл. Например, "16.07.2009 0:29:31".
_Добавить текст Данное комманда дописывает в указанный вами файл текст или данные. Если же вы хотите полностью переписать файл, воспользуйтесь коммандой Сохранить в файле
_Зашифрованный Данное свойство показывает, является ли указанный в скобках файл зашифрованным или нет. Зашифрованные файлы может просматривать только пользователь учетной записи Windows, в которой файлы были зашифрованы. В остальных учетных записях такие файлы не откроются.
_Зашифровать комманда шифрует файл для того чтобы другие учетные записи Windows и другие компьютеры не могли прочитать этот файл. При этом данный пользователь учетной записи Windows может читать этот файл свободно.
_Количество папок
_Количество файлов
_Копировать Данная комманда копирует файл или папку.
_Не индексируется Данное свойство показывает, является ли указанный в скобках файл не индексируемым или нет.
_Определить без расширения Данное свойство определяет имя файла без расширения из переданного ему пути до файла или папки. Например для файла "C:/Program Files/File.jpg", это свойство вернет "File".
_Определить имя папки Данное свойство определяет имя родительской папки для файла или директории. Например, для файла "C:/Program Files/File.jpg" или для папки "C:/Program Files/Алгоритм 2", это свойство вернет "Program Files".
_Определить имя файла Данное свойство определяет имя файла из переданного ему пути до файла или папки. Например для файла "C:/Program Files/File.jpg", это свойство вернет "File.jpg". Для папки "C:/Program Files/Алгоритм 2", это свойство вернет "Алгоритм 2".
_Определить корневую Данное свойство определяет корень - диск для переданного пути. Например для файла "C:/Program Files/File.jpg", это свойство вернет "C:/".
_Определить размер файла Данное свойство определяет размер файла в байтах.
_Определить расширение Данное свойство определяет путь до файла или папки. Например для файла "C:/Program Files/File.jpg", это свойство вернет "C:/Program Files".
_Определить родительскую Данное свойство определяет путь до файла или папки. Например, для файла "C:/Program Files/File.jpg" или для папки "C:/Program Files/Алгоритм 2", это свойство вернет "C:/Program Files".
_Открыть папку
_Открыть файл Данное свойство открывает файл и получает всё его содержимое.
_Папка Данное свойство показывает, является ли указанный в скобках путь папкой или нет.
_Переименовать
_Переместить Данная комманда перемещает файл или папку.
_Поиск файлов Данное свойство ищет файлы и папки в указанной ему директории. При поиске можно пользоваться шаблонами поиска - "*","?","+". Например, "a*.jpg" - найдет все файлы с расширением jpg, начинающиеся на "а" (например, "аbcdef.jpg", "a2.jpg", "a.jpg"). "a?c.exe" - найдет все приложения, состоящие из трех букв, начинающиеся на "а" и кончающиеся на "с" (например, "аbc.exe", "a2c.exe")
_Получить диски Данное свойство возвращает список файлов из указанной папки. Файлы в списке перечислены через "; " и выглядят примерно так: "A:/; C:/; D:/".
_Получить папки Данное свойство возвращает список папок из указанной директории. Папки в списке перечислены через "; " и выглядят примерно так: "C:/Program Files; C:/Windows; C:/Temp".
_Получить файлы Данное свойство возвращает список файлов из указанной папки. Файлы в списке перечислены через "; " и выглядят примерно так: "C:/1.jpg; C:/2.bmp; C:/file.exe".
_Расшифровать комманда расшифровывает зашифрованный ранее файл. Зашифрованный файл другие учетные записи Windows и другие компьютеры не могут прочитать. При этом пользователь учетной записи Windows, в которой был зашифрован файл, может читать его без проблем. Также нельзя расшифровать файлы, зашифрованные в других учетных записях Windows.
_Системный
_Скрытый Данное свойство показывает, является ли указанный в скобках файл скрытым или нет.
_Создать папку комманда создает папку на компьютере. Причем, например, папки "С:/a/b/c" успешно создадутся, даже если до этого не существовало даже папки "С:/a".
_Сохранить в файле Данное комманда сохраняет в указанном вами файле текст или данные. Если файл содержал данные, они полностью заменятся на новые. Если же вы хотите дописать файл, воспользуйтесь коммандой Добавить текст
_Сохранить рисунок Данное комманда сохраняет в указанном вами файле рисунок.
_Существует папка Данное свойство показывает, существует ли данная папка на компьютере или нет.
_Существует файл Данное свойство показывает, существует ли данный файл на компьютере или нет.
_Только для чтения Позволяет узнать является ли файл только для чтения или позволяет сделать его таковым.
_Удалить Данная комманда удаляет файл или папку безвозвратно.
