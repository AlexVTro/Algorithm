Описание

Раздел свойств позволяет настроить все свойства главных действующих лиц программ - Объектов (кнопки, надписи, окна).

Свойства это настройки объектов (цвет, текст, размер), изменяя которые вы можете настроить объект необходимым образом. Подробнее узнать про свойства каждого объекта вы можете в разделе Объекты.

Для того чтобы изменить свойство, достаточно выбрать его из списка, отредактировать значение и нажать Enter.
