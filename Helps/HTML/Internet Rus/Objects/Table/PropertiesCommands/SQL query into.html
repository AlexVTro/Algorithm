Описание

комманда выполняет запрос типа INSERT, UPDATE, DELETE, изменяя таким образом данные в базе данных. Подробнее об SQL: http://sitemaker.ru/technologies/database/sqlqueries/

Данные, требуемые для выполнения свойсва или комманды:

Объекты.Таблица.SQL запрос изменения (Тип базы данных, Путь к файлу, SQL запрос изменения базы данных)

Данные
Описание
Тип базы данных База данных - организованное хранилище информации. Алгоритм поддерживает пока два типа баз данных: Access и Excel.
Путь к файлу Тут нужно указать путь к файлу. Путь должен обозначаться в кавычках, например "C:\1.jpg" или "\2.jpg" (во втором случае файл "2.jpg" должен располагаться в папке с проектом)
SQL запрос изменения базы данных SQL запрос, направленный на изменение данных в базе данных (запросы INSERT, UPDATE, DELETE), например "DELETE * FROM table_name"
