Найти номер записи

Описание

Возвращает номер записи списка по ее значению. Если введем "Запись1" - вернет значение 0, "Запись2" - значение 1 (поскольку номерация начинается с нуля).

Данные, требуемые для выполнения свойсва или комманды:

Объекты.Список галочек.Найти номер записи (Запись из списка)

Данные
Описание
Запись из списка Текст записи, номер которой вы хотите найти.
