Описание

Возвращает номер строки, которая располагается перед текущей записью в области выборки. Если, например, до этого запрашивали последнюю строку, то сейчас комманда выдаст пред-последнюю. В следующий раз пред-пред-последнюю и т.д.

Данные, требуемые для выполнения свойсва или комманды:

Объекты._Объект события.Номер предыдущей строки (Номер начальной строки, Область выборки)

Данные
Описание
Номер начальной строки Строка, с которой начинают выборку. Первая строка имеет номер "0".
Область выборки Указывает где искать строку. Например если область выборки "Выделенные строки" то данная функция будет видеть только выделенные строки и искать только в них.
