Описание

С помощью этого свойства можно запретить редактирование любых строк таблицы.

Данные, требуемые для выполнения свойсва или комманды:

Объекты._Объект события.Строка только для чтения (Номера строк через запятую)

Данные
Описание
Номера строк через запятую Перечислите нужные строки через запятую в двойных кавычках, например, "0,1,8,3" или "2".
