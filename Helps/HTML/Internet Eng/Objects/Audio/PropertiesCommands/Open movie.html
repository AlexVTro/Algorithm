Open movie
Help -> Objects -> Audio -> Properties and commands -> Open movie

See also, Sample
Collapse all

Description

This property is used to select a file for playing. Only after the file has been opened it can be played with the help of the property Play movie.

The data required to implement properties or commands:

Objects.Audio.Open movie (The path to the file)

Data
Description
The path to the file Тут нужно указать путь к файлу. Путь должен обозначаться в кавычках, например "C:\1.jpg" или "\2.jpg" (во втором случае файл "2.jpg" должен располагаться в папке с проектом)

Remark

Example

Example of how mp3-file is opened and played:

Algorithm2
[Copy to clipboard] 
Window1.Audio1.Open movie = "С:\Track1.mp3"
Window1.Audio1.Play movie
