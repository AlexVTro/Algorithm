Description

It is an image which locates on the object. It is behind a text, but over a background image.

Image is specified by setting path to the file of this image.

Remark

The path to the image can be set as absolute, i.e. it starts with the drive letter:
"C:/Images/pic.jpg"
or it can be relative and starts from the folder of your project:
"/Images/pic.jp"

Example

Variants of how a main window image can be set:

Algorithm2
[Copy to clipboard] 
Window1.Window1.Image = "C:/1.bmp"
Window1.Window1.Image = "/1.bmp"
Window1.Window1.Image = My documents & "/2.jpg"
Window1.Window1.Image = Window1.Button1.Background image
