Text align
Help -> Objects -> Button -> Properties and commands -> Text align

See also, Sample
Collapse all

Description

It defines a text position on the object (On left, On bottom, Centered).

All available positions are described in Quick fill words -> Alignments.

Example

Variants of how a text can be positioned on the button

Algorithm2
[Copy to clipboard] 
Window1.Button1.Text align = Center
Window1.Button1.Text align = "Bottom left"
Window1.Button1.Text align = Window1.Button2.Image align
Window1.Button1.Text align = Window1.Text1.Text
