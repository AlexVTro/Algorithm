Back color

Collapse all

Description

It is the main color of the object.

All colors can be specified in words (red, blue, black). Learn more about them in the section Quick fill words -> Colors. Or you can specify a color in a special form where you input a tint of the red color, then in semi-colon the tint of the green color and again in semi-colon the tint of the blue color. Maximum intensity is 255, minimum is 0. For example: 

"255; 0; 0;" = red
"0; 0; 255;" = blue
"0; 0; 0;" = black
"255; 255; 255;" = white
"255; 255; 128;" = pale yellow

Remark

If you want to make a color transparent (not all objects has this function) then specify for its property the value "Nothing" or "".

Example

Variants of how main window color can be set:

Algorithm2
[Copy to clipboard] 
Window1.Window1.Color = red
Window1.Window1.Color = "255; 201; 128;"
Window1.Window1.Color = "128;128;0;"
Window1.Window1.Color = Window1.Button1.Fore color
