Description

Using this object you can turn your program into a real ShareWare program. Its function is to watch over the trial period, activation, registration key and so on.

Trial makes the registration of the program self-acting. You can find the example of how this object is used in Examples\Complex projects\Trial.

The object gives no guarantee that your ShareWare program won't be hacked. Be aware that the hacker can do it easily. That's why we advise you to supplement the object Trial with your own methods of watching over the trial period and activation so that it would become complicated for the hacker to hack your program.

Features of the object:
Events Events are actions performed by the user (clicking, cursor moving, keys pressing) which can be registered by objects.
Properties and Commands Properties are settings of objects (color, text, size). By changing them you can set the object in a way you need. Objects can also execute some actions which are called commands and relate to them. You can change properties manually in Properties panel or program their changes in Actions tab. Remark

Every object Trial is associated with your program only owing to the property Id program (unique identifier of your program) and the property Key encryption (contains the algorithm of encryption). So if you use several Trials and want them to work with the same keys and Id, then make sure that they have identical Id program and key encryption!
