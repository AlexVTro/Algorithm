Image align
Help -> Objects -> Url -> Properties and commands -> Image align

See also, Sample
Collapse all

Description

The image position (alignment) on the object (On left, On bottom, Centered, etc.).

All available alignments are described in: Quick fill words -> Alignments.

Example

Variants of how button image alignment can be set:

Algorithm2
[Copy to clipboard] 
Window1.Button1.Image align = Center
Window1.Button1.Image align = "Bottom left"
Window1.Button1.Image align = Window1.Button2.Text align
Window1.Button1.Image align = Window1.Text1.Text
