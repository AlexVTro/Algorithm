Description

Events are actions performed by the user (clicking, cursor moving, keys pressing) which can be registered by objects.

Every object has particular events to register. You can program registering of events in the Actions tab. After you've programmed it all actions inside of the event in the Actions tree would be executed only when the given event has occured (for example, after clicking).

Can go back changedFor the object Browser the option go back can be set or can be not. This event is called when this setting is changed.
Can go forward changedFor the object Browser the option go forward can be set or can be not. This event is called when this setting is changed.
Document completedThe event is called when a web-page has been completely loaded and the browser displays it. The object where this event is called gets new options.
File downloadThe event is constantly being called while a web-page is being downloaded into the browser.
Got focus The event is called when the object gets focus (it becomes active).
Lost focus The event is called when the object loses focus (i.e. some other object becomes active).
NavigatedThis event is called when the user jumped from url and the page has been loaded. As a result the object where this event was called gets additional options.
NavigatingThe event is called before jumping to a new web-page. As a result the object where the event was called gets additional options.
New window The event is called when the page is opened in a new window. As a result the object where the event was called gets additional options.
Size changedThe event is called after the object's sizes have changed (width, height).
Status text changedStatus text is constantly changing depending on what is going on in browser now. This event is executed after any change has been done.
