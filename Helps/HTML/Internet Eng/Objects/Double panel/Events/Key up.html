Description

The event is called for the active object (the object is in focus) when the user presses a keyboard key. Then this active object gets additional options.

Additional features which appear at the sites at this event:

Object that happened on the event, adds a happy new features (for example, X and Y mouse, key pressed). These properties can not be changed, but you can see them (because they can not be changed, then find the properties can be Wizard complex action ). Below are all the properties of the event.

Property
Description
_Key pressed Contains the name of keyboard key (а, b, ctrl, enter) which is pressed during the event.
_Keyboard alt Displays whether the alt-key (Alt) is pressed at the moment or not.
_Keyboard control Displays whether the control-key (Ctrl) is pressed at the moment or not.
_Keyboard shift Displays whether the shift-key (Shift) is pressed at the moment or not.
