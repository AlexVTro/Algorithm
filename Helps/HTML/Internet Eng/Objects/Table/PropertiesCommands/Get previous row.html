Get previous row
Help -> Objects -> Table -> Properties and commands -> Get previous row

See also
Collapse all

Description

Returns the number of a row which goes before the current row in the area of selection. If the last row was returned before the command would return now the last but one row. Next time it'll return the row previous to that row which is last but one row and so on.

The data required to implement properties or commands:

Objects.Table.Get previous row (Number first lines, Field sampling)

Data
Description
Number first lines Строка, с которой начинают выборку. Первая строка имеет номер "0".
Field sampling Указывает где искать строку. Например если область выборки "Выделенные строки" то данная функция будет видеть только выделенные строки и искать только в них.
