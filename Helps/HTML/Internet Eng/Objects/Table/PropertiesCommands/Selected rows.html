Description

The property selects table rows or indicates which rows are selected. The output is given in the form of 0,2,3, i.e. the selected rows are the first, third and forth rows (numbering begins from "0"; 0 - is the first row, 1 - is the second one, etc.).

To select a block of cells this command should be combined with the command Selected columns: 

Window1.Table1.Selected rows = "1,0"
Window1.Table1.Selected columns = "1"
This fragment of code selects the first and second cells of the second column.
