Items index of
Help -> Objects -> Checked list -> Properties and commands -> Items index of

See also, Sample
Collapse all

Description

Returns the index of a list item in accordance with its value. If you input "Record1" it returns "0", for "Record2" it returns "1" (as numbering begins from 0).

The data required to implement properties or commands:

Objects.Checked list.Items index of (Recording from the list)

Description
