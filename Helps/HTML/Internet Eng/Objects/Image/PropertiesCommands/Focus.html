Description

Focus makes one particular object active, so the user works exactly with it (e.g. the user can focus a text box and values of all buttons he presses are input into this text box).

The concept "focus" has the same meaning as in photocameras - "be in focus" means "be at the center of attention". In our case - be at the center of the user attention. If an object is out of the focus it remains enabled, but the user does not use it yet.
