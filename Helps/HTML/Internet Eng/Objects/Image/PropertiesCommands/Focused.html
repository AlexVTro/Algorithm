Description

If Focused is "Yes" the object, the user works with, becomes active (e.g. the user inputs data into a text box).

The concept "focus" has the same meaning as in photocameras - "be in focus" means "be at the center of attention". In our case - be at the center of the user attention. If this property is "No" the object remains enabled, but the user does not use it yet.
