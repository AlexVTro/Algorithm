Background image
Help -> Objects -> Image -> Properties and commands -> Background image

See also, Sample
Collapse all

Description

It is a background image of the first part of Double page.

You can set this image by specifying the path to a file of a image.

Remark

The path can be absolute, i.e. it begins from the letter of the drive
"C:/Images/pic.jpg"
and relative. In this case the path begins from the folder where the project locates.
"/Images/pic.jpg"

Example

Variants of how Double panel background image can be set:

Algorithm2
[Copy to clipboard] 
Window1.Double panel1.Background image1 = "C:/1.bmp"
Window1.Double panel1.Background image1 = "/1.bmp"
Window1.Double panel1.Background image1 = My documents & "/2.jpg"
Window1.Double panel1.Background image1 = Window1.Кнопка1.Background image
