Font family

Collapse all

Description

It is the name of the font used for the object text (Arial, Time new roman, etc.).

All available fonts are specified in Quick fill words -> Font family.

Example

Variants of how button font can be set:

Algorithm2
[Copy to clipboard] 
Window1.Button1.Font family = Arial Black
Window1.Button1.Font family = "Time" & " new roman"
Window1.Button1.Font family = Window1.Button2.Font family
Window1.Button1.Font family = Window1.Text1.Text
