Description

It is a symbol the mouse pointer is changed to when it is moved over the object.

Every cursor symbol has its own name. All cursor names you can find in Quick fill words -> Cursors

Example

Variants of how a cursor can be set (after it has been moved over a button):

Algorithm2
[Copy to clipboard] 
Window1.Button1.Cursor = "Pan North"
Window1.Button1.Cursor = Hand
Window1.Button1.Cursor = Window1.Window1.Cursor
Window1.Button1.Cursor = Window1.Text1.Text
