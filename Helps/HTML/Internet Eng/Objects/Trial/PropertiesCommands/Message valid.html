Description

When the user has successfully activated the program, the corresponding message is displayed to the user. This property stores the text of the message.

Remark

If it is necessary to input quotes in this message, then you should double them in the text:
"There a text is without quotes and ""there is in quotes""."

To insert Newline (equal to pressing of Enter key) use Quick fill words -> Others or input a symbol "¶"

Window1.Button1.Text = "Text ¶ Text on a newline"
