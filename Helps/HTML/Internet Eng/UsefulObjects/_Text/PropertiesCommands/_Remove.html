Description

Removes a certain part of the text from it.

The data required to implement properties or commands:

_Useful objects._Text._Remove (Original text, Character number beginning piece, Length of piece)

Data
Description
Original text Текст оригинал, который будет использоваться данным свойством или коммандой.
Character number beginning piece Номер символа, с которого надо начинать брать кусок текста. Первый символ имеет номер "1", второй "2" и т.д.
Length of piece Количество символов какой длинны будет кусок текста.
