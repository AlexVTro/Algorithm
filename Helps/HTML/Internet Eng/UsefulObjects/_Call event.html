Description

Using this useful object you can call any event of the program (click, text input, mouse movement) independently of the user actions.

Example:

There Double click on the Button2 initiates actions of absolutely other event - of Click on the Button1. Thus the same actions are executed both on click on the Button1 and on double click on the Button2 (in the example the text of the Label1 is changed). This is a very convenient method in cases there are a lot of different buttons executing the same actions.
