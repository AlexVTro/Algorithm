Description

Is used to know whether a file is read-only or not and in case the file is not it can turn it into read-only.

The data required to implement properties or commands:

_Useful objects._Files and paths._Read only (The path to the file or folder, Attribute value)

Data
Description
The path to the file or folder Тут нужно указать путь к файлу или папке. Путь должен обозначаться в кавычках, например "C:\1.jpg" или "\2.jpg" (во втором случае файл "2.jpg" должен располагаться в папке с проектом).
Attribute value Указывает, установлено ли данное свойство у файла (значение "Да"), либо нет (значение "Нет").
