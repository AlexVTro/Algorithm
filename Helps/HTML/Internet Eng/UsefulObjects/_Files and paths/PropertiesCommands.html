Description

Properties are settings of objects (color, text, size). By changing them you can set the object in a way you need. Objects can also execute some actions which are called commands and relate to them.

You can change properties manually in Properties panel or program their changes in Actions tab. In the second case in the Events tab the event is created for the object and under this event the given property is changed. Examples of how to use properties and commands in Actions tab:

Window1.Object1.Text = "New text1"
Window1.Object2.Image = "C:\Pic.jpg"
Window1.Object2.Focus

List of object properties _Files and paths:
_Access time This property indicates or changes the date and time when the file was addressed last. For example, "16.07.2009 0:29:31".
_Append text This command is used to append text or data into the specified file. If you want to overwrite a file, then use the command _Save in file
_Archive This property indicates whether a file in brackets is an archive or not.
_Copy The command copies a file or a folder.
_Create directory The command is used to create any directory in the system. So, folders of the path "С:/a/b/c" would be successfully created even if there was no directory "С:/a" before.
_Create time The property indicates or changes the time of file creation. For example: "16.07.2009 0:29:31".
_Decrypt The command decrypts an encpypted file. The ecrypted file cannot be decrypted in other accounts of Windows or on the other computer. Only the user of the Windows account, where the file was encrypted, can decrypt it easily.
_Delete This command deletes a file or a folder irretrievably.
_Directories count
_Edit time The property indicates or changes the time when the file was edited last. For example, "16.07.2009 0:29:31".
_Encrypt The command encrypts a file so that the file could not be decrypted in other Windows accounts and on other computers. And the user of the Windows account where the file was created could open this file freely.
_Encrypted The property indicates whether a file specified in brackets is encrypted or not. Encrypted files can be viewed only by the user of that Windows account where files were encrypted. In other accounts these files are not accessible.
_Exist file This property indicates whether the file exists on the computer or not.
_Exist path The property indicates whether the path exists on the computer or not.
_Files count
_Find file The property searches files and folders in the specified directory. For searching you can use a pattern - "*","?","+". E.g. if you specify "a*.jpg", it would find all files with extension jpg beginning with "a" (for example, "аbcdef.jpg", "a2.jpg", "a.jpg"). For "a?c.exe" it would find all applicaions consisting of three letters beginning with "a" and ending with "c" (for example, "аbc.exe", "a2c.exe").
_Folder The property indicates whether the path specified in brackets is a folder or not.
_Get drives This property returns a list of drives. Drives are enumerated in "; " and look like: "A:/; C:/; D:/".
_Get extension The property defines the path to a file or a folder. For example, for the file "C:/Program Files/File.jpg", this property returns "C:/Program Files".
_Get file name The property returns the name of a file located in specified path. E.g. for the file "C:/Program Files/File.jpg" it returns "File.jpg". For the folder "C:/Program Files/Algoithm2" it returns "Algoithm2".
_Get file name without extension The property returns the name of a file without extension located in the specified path. For the file "C:/Program Files/File.jpg" it returns "File".
_Get file size The property defines the file size in bytes.
_Get files The property returns a list of files from the folder specified. Files in the list are enumerated in "; " and it looks like: "C:/1.jpg; C:/2.bmp; C:/file.exe".
_Get parent The property defines a path to a file or folder. E.g. for the file "C:/Program Files/File.jpg" or for the folder "C:/Program Files/Algorithm2" this property returns "C:/Program Files".
_Get path name This property defines the name of a parent path for specified file or folder. E.g. for the file "C:/Program Files/File.jpg" or for the folder "C:/Program Files/Algorithm2" this property returns "Program Files".
_Get paths This property returns a list of folders from the specified directory. Folders are enumerated in "; " and the list looks like: "C:/Program Files; C:/Windows; C:/Temp".
_Get root The property defines the root - a drive for a specified path. E.g. for the file "C:/Program Files/File.jpg" this property returns "C:/".
_Hider This property indicates whether a file specified in brackets is hidden or not.
_Move The command moves a file or a folder.
_Not indexer The property indicates whether a file in the brackets is indexable.
_Open directory
_Open file The property opens a file and gets all its contents.
_Read only Is used to know whether a file is read-only or not and in case the file is not it can turn it into read-only.
_Rename
_Save in file The command saves a text or data into the specified file. If the file contains data it is replaced comletely by a new one. If you want just to append the file then use the command _Append text
_Save picture The command saves a picture into the specified file.
_Sys
_Temp This property indicates whether a file in brackets is a temporary file or not.
