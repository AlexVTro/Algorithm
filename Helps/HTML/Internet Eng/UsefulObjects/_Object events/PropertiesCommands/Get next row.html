Get next row
Help -> Objects -> _Object events -> Properties and commands -> Get next row

See also
Collapse all

Description

Returns the number of a row which goes after the current row in the area of selection. If the first row was returned before the command would return the second row now. Next time it'll return the third row and so on.

The data required to implement properties or commands:

Objects._Object events.Get next row (Number first lines, Field sampling)

Data
Description
Number first lines Строка, с которой начинают выборку. Первая строка имеет номер "0".
Field sampling Указывает где искать строку. Например если область выборки "Выделенные строки" то данная функция будет видеть только выделенные строки и искать только в них.
