Description

Returns the number of a row which is the last in the area of selection.

The data required to implement properties or commands:

Objects._Object events.Get last row (Field sampling)

Data
Description
Field sampling Указывает где искать строку. Например если область выборки "Выделенные строки" то данная функция будет видеть только выделенные строки и искать только в них.
