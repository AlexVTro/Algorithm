Description

A text specified in this property is displayed on default in the text field for activation key input.

Remark

If it is necessary to input quotes into text then you should double them inside of the text:
"There a text is without quotes and ""there is in quotes""."

To insert Newline (equal to pressing of Enter key) use Quick fill words -> Others or input a symbol "¶"

Window1.Button1.Text = "Text ¶ Text on a newline"
