Description

Objects (buttons, windows, images) are blocks which are used for creation of any program.

Objects can register user's actions (which are called events), such as click, moving of a mouse on some object, changing of a text and so on. To set registering of any action use Events tab.

Objects can be set in a way you need by using a great number of properties. Besides, you can change properties while the program is functioning by setting the algorithm in Actions tab. These algorithms which change properties are programming.

The list of all objects:
AudioThe object plays a wave file (e.g. music, sounds) which is specified in the property File name played.
ButtonThe main function of this object is to register clicks performed on the object.
CalendarCalendar helps to set the date.
CheckUsing this object you can make an option for the user to choose out of several options or variants of answers.
Checked listUsing this object you can make an option for the user to select out of several variants. The user simply puts checks on necessary variants.
ClientServerThis object makes it possible to exchange alphanumeric data (values), download and transmit files. This flexible tool will help you to create your own ICQ-clients, transmit and download different files from a computer to a computer, etc.
Color dialog In Color dialog you can set any color. The dialog can be opened by the command Show dialog and the color specified by user is to be displayed in the property Dialog color.
Combo boxUsing this object you can make an option for the user to choose several options or variants of answers.
Context menuThis menu usually opens when the user clicks the right button of the mouse. It consists of Menu items.
Double panelThis is a usual Panel, but it consists of two parts - Panel1 and Panel2.
Font dialog In Font dialog you can make an option for the user to select out of different fonts. The window can be opened by the command Show dialog and the font selected by the user is to be specified in Font family, Font size, etc.
ImageThe main function of this object is to display an image specified by you.
InternetThe object makes it possible to interact with the Internet, make POST/GET–requests, get the results of requests, download files from the Internet. With the help of POST/GET-requests you can get the contents of web pages and also post user data (e.g. to substitute Login and Password or other information into variables). The majority of properties return headers parameters received from the server. So when you've learned them, many of your questions would put out of your mind. To analyse the object better see the example "Internet" in Projects\Examples\Simple Examples and also read the web page of Wikipedia
ItemItems are what Menu, Context menu and Tool panel consist of.
LabelThe main function of the object is to display a text specified by you.
List boxUsing this object you can make an option for the user to select out of several options or variants of answers.
MediaThe object plays video and audio files (e.g. films, music) specified in the property File name played.
MemoryThe object saves data specified for the storage in the property Value.
MenuThis object is the main menu which locates at the top of the application window. It consists of Menu items.
Open dialog Using Open dialog you can make an option for the user to select any file. The window can be opened by the command Show dialog and the file specified by the user is to be displayed in the property File name.
Panel On Panel you place other objects (buttons, text boxes, labels and other panels).
Path dialog Using Path dialog you can make an option for the user to select any path. The window can be opened by the command Show dialog and the path specified by the user is to be displayed in the property Selected path.
Print dialogPrint dialog helps to organize printing of a text file or an image. The window can be opened by the command Show dialog.
Progress barThe main function of the object is to display the process of downloading/executing (e.g. downloading of a file from the Internet, loading of a program, etc.).
RadioUsing this object you can make an option for the user to select only one option out of several.
Rich documentThis is an extended version of the object Text. Rich document is used for editing color, size and font of every letter, for inserting images and for doing a lot of other things.
Save dialog Using Save dialog you can make an option for the user to select a file. The window can be opened by the command Show dialog and the file specified by the user is to be displayed in the property File name.
StopwatchStopwatch is used for counting the time elapsed or for executing actions with the definite periodicity (for example, in every several seconds).
Tab pageThis object is enclosed in the object Tab pages and is used for arranging of other objects (buttons, text boxes, labels) on it.
Tab PagesThis object consists of Tab pages with the help of which you can group different objects (buttons, text boxes, labels).
TableThe Table is a very practical method of displaying information in the form of rows and columns.
TextThe main function of this object is to get any text from the user of the program. Use the property Text to get the text entered.
Tool panelTool panel is a set of different buttons collected on one panel. These buttons are called panel items.
Track barFeatures of the object:
TrialUsing this object you can turn your program into a real ShareWare program. Its function is to watch over the trial period, activation, registration key and so on.
UrlThe main function of the object is to display the text with url after clicking which the user jumps, for example, to the web page.
Web browserThe object opens web pages (html-documents) which are specified in the property Url
WindowThis is the main object of the program. Windows contain all other objects.
