Description

You can open a configuration dialog box by selecting menu items Tools -> Settings. Dialog box sections:

Interface language:
Here you can choose a language of development environment. All menus and titles will be shown in the language specified, but the language in which the project has been created initially remains for all actions and programming.

Programming language:
is the language in which all new projects will be programmed after the given option has been changed, but the language in which the current project has been created initially remains for all actions and programming of this project. To change the language of the current project select menu items Tools -> Translate project into.

Skin:
is a theme of Algorithm2 development environment.

Project Folder:
is a folder where all projects are stored on default. You can create your own directories there.

Default folder with project images:
When you specify images in the Properties panel they are automatically copied into a special image folder which is stored in the directory of your project. You can rename this image folder in this setting and it will have this title in all new projects, but for the current project it won't change.

Current project's images folder:
When you specify images in the Properties panel they are automatically copied into a special image folder which is stored in the directory of your project. You can rename this image folder in this setting and the title will be changed right for the current project.

Default image filename:
When you specify images in the Properties panel they are automatically copied into a special image folder which is stored in the directory of your project. The image title changes to the title specified in this setting.

Program's icon:
When you create an exe-file of your project (File -> Compile Executable) the file gets an icon specified in this option.

Show loading window at the project start:
When users run your program it takes some time for the program to be loaded completely and to be displayed. So during this time a loading window with a progress bar is being displayed to the user. In this setting you can disable the option.
