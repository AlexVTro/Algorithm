Description

In the Actions tree all software contents of you program are created. All objects, events, actions, conditions, cycles, comments are contained there.

Structure of the actions tree:

The first, fundamental branches are Windows. These branches include all objects of the given window. Then there are Events of each object. Events include in their turn all actions, conditions and cycles. Comments can be located in any part of the actions tree.

You can easily relocate, copy, insert, delete all elements of the actions tree. Besides, you can edit them in two ways:
1) Select a necessary action and edit it in the text box which is placed below the actions tree.
2) Select a necessary action and click the right button of the mouse, then select menu item Edit (or simply click twice on the action). The action will appear in the actions panel so you can edit it now.

Besides you can flag all actions with checkpoints:

All information on checkpoints and flagging you can find in the section Project debugging
