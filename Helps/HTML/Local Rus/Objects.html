Описание

Объекты (кнопки, окна, рисунки) это блоки, из которых и создаются все программы

Объекты могут отлавливать действия пользователя (они называются события), такие как например клик, движение мышкой по объекту, изменение текста и т.д. Чтобы поставить отлов какого-нибудь действия используйте Раздел событий

Объекты можно настроить как требуется при помощи большого количества свойств. Кроме того, свойства можно изменять в процессе работы программы, задав правила их изменения в Разделе действий. Эти правила изменения свойств и являются программированием.

Список всех объектов:
АудиоОбъект воспроизводит звуковой файл (например музыку, звуки), который ему укажут в свойстве Файл проигрывания
БегунокБегунок имеет перемещающийся элемент, который передает свое положение, куда его передвинул пользователь.
Большой списокС помощью этого объекта можно предоставить пользователю выбор из нескольких вариантов ответа.
БраузерОбъект открывает интернет-страницы (html-документы), которые указаны в свойстве Ссылка
Всплывающее менюЭто меню, которое обычно вызывается кликом правой кнопки мыши. Оно состоит из Пунктов меню.
ГалочкаС помощью этого объекта можно предоставить пользователю выбор каких-нибудь нескольких опций.
Двойная панельЭто обычная Панель, но она состоит из двух частей - Панель1 и Панель2.
ЗакладкаЭто объект, который входит в состав объекта Закладки, и позволяет размещать на себе другие объекты (кнопки, текстовые поля, надписи)
ЗакладкиЭто объект, состоящий из Закладок, с помощью которых можно удобно группировать разные объекты (кнопки, текстовые поля, надписи)
ИнтернетОбъект дает возможность работать с интернетом, осуществлять POST/GET –запросы, получать результаты запросов, скачивать файлы с ресурсов сети Интернет. С помощью POST/GET-запросов можно получать(GET) содержимое web-страниц, а также отправлять (POST) пользовательские данные какому-нибудь ресурсу Интернет (например, подставлять в переменные Логин и Пароль, или другую информацию). Большенство свойств возвращают параметры хидеров(header), полученных от сервера. Потому, разобравшись с ними, вопросов у Вас будет намного меньше. Для более понятного изучения объекта просмотрите пример "Интернет" в папке Проекты\Примеры\Простые примеры, а также прочитайте страничку Википедии
КалендарьКалендарь предоставляет возможность пользователю удобно выбрать дату.
КлиентСерверЭто объект с помощью кторого можно обмениваться текстовыми данными (значениями), скачивать и передавать файлы. Этот гибкий инструмент позволит Вам создавать свои собственные ICQ клиенты, передавать, скачивать с компьютера на компьютер различные файлы и тому подобное.
КнопкаОсновное предназначение этого объекта - улавливать клики по нему.
МедиаОбъект воспроизводит видео и звуковые файлы (например фильмы, музыку), который ему укажут в свойстве Файл проигрывания
МенюЭто главное меню, которое располагается в верху окна. Оно состоит из Пунктов меню.
НадписьОсновное предназначение этого объекта - отображать текст, который вы ему укажите.
ОкноЭто главный объект. На окнах располагаются все остальные объекты.
Окно открытияОкно открытия позволяет организовать выбор какого-нибудь файла пользователем. Отображение окна вызывается коммандой Запустить окно, а выбранный пользователем файл будет в свойстве Имя файла.
Окно папокОкно папок позволяет организовать выбор какой-нибудь папки пользователем. Отображение окна вызывается коммандой Запустить окно, а выбранная пользователем папка будет в свойстве Выбранная папка.
Окно печатиОкно печати позволяет организовать печать текстового файла или рисунка. Отображение окна вызывается коммандой Запустить окно.
Окно сохраненияОкно сохранения позволяет организовать выбор какого-нибудь файла пользователем. Отображение окна вызывается коммандой Запустить окно, а выбранный пользователем файл будет в свойстве Имя файла.
Окно цветаОкно цвета позволяет организовать выбор какого-нибудь цвета пользователем. Отображение окна вызывается коммандой Запустить окно, а выбранный пользователем цвет будет в свойстве Выбранный цвет.
Окно шрифтаОкно шрифта позволяет организовать выбор шрията пользователем. Отображение окна вызывается коммандой Запустить окно, а выбранный пользователем шрифт будет в свойстах Шрифт, Шрифт размер и т.д..
ПамятьОбъект сохраняет данные, которые вы ему скажите хранить в свойстве Значение.
ПанельПанель предназначена для размещения не ней других объектов (кнопки, текстовые поля, надписи, другие панели)
Панель инструментовПанель инструментов это набор различных кнопок, собранных в одной панели. Эти кнопки называются Пункты панели.
Полоса загрузкиДанный объект служит для визуального отображения загрузки/выполнения какого-либо процесса (скачки файла из интернета, загрузки программы, выполнения процесса и т.д.)
ПунктИз Пунктов состоят Меню, Контекстное меню, Панель инструментов.
РисунокОсновное предназначение этого объекта - отображать рисунок, который вы ему укажите.
СекундомерС помощью Секундомера можно измерять пройденное время или выполнять действия с определенной периодичностью, например в несколько секунд.
СписокС помощью этого объекта можно предоставить пользователю выбор из нескольких вариантов ответа.
Список галочекС помощью этого объекта можно предоставить пользователю выбор из нескольких вариантов ответа. Нужные ответы пользователь может отметить галочками.
СсылкаОсновное предназначение этого объекта - отображать текст со ссылкой, на которую можно нажать и перейти по ней, например на интернет-страницу.
ТаблицаТаблица это очень удобный способ отображения информации в виде Строк и Столбцов.
ТекстОсновное предназначение этого объекта - получение какого-нибудь текста от пользователя программы. Получеить введенный текст можно с помощью свойства Текст.
Текстовый документЭто расширенная версия объекта Текст. Текстовый документ позволяет редактировать цвет, размер, шрифт каждой отдельной буквы, вставлять рисунки и много другое.
ТочкаС помощью этого объекта можно предоставить пользователю выбор какой-нибудь одной опции из нескольких.
ТриалС помощью этого объекта можно сделать из вашей программы полноценную ShareWare. Он возьмет на себя все заботы об отслеживании триального периода, об активации, о ключе регистрации и т.д.
