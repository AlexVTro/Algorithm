Описание

Значек, на который заменяется указатель мыши, при наведении курсора на объект.

У каждого значка курсора есть свое название. Все названия курсоров можно найти в Вспомогательные слова -> Курсоры

Пример

Варианты изменения курсора у кнопки

Aлгоритм2
[копировать в буфер обмена] 
Окно1.Кнопка1.Курсор = "Направление север"
Окно1.Кнопка1.Курсор = Рука
Окно1.Кнопка1.Курсор = Окно1.Окно1.Курсор
Окно1.Кнопка1.Курсор = Окно1.Текст1.Текст
