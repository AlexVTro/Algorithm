Описание

Возвращает номер столбца, на который попадают переданные координаты X и Y. Координаты можно взять, например, из свойств _Мышь X, _Мышь Y.

Данные, требуемые для выполнения свойсва или комманды:

Объекты._Объект события.Номер столбца по координатам (X, Y)

Данные
Описание
X Координата X относительно объекта. Отсчет идёт слева на право, т.е. в левом краю объекта X=0, а на правом X максимален (равен Ширине объекта)
Y Координата Y относительно объекта. Отсчет идёт сверху вниз, т.е. в верхнем краю объекта Y=0, а в самом низу Y максимален (равен Вышине объекта)
