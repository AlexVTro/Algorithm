Описание

Когда пользователь успешно активирует программу ему выведется сообщение. Текст этого сообщения хранит это свойство.

Примечание

Если в сообщении требуется записать кавычки, то это делается двойным повторением кавычек внутри текста, который вы хотите присвоить:
"Вот идет текст без кавычек, а ""вот в кавычках""."

Для того чтобы вставить символ новой строки (аналог нажатия клавиши Enter) используйте Вспомогательные слова -> Прочие или вставьте символ "¶"

Окно1.Кнопка1.Текст = "Текст ¶ Текст на новой строке"
