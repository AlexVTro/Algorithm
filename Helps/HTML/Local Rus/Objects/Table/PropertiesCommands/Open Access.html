Описание

комманда открывает базу данных формата Access, и отображает указанную таблицу из базы.

Данные, требуемые для выполнения свойсва или комманды:

Объекты.Таблица.Открыть Access (Путь к файлу, Название таблицы)

Данные
Описание
Путь к файлу Тут нужно указать путь к файлу. Путь должен обозначаться в кавычках, например "C:\1.jpg" или "\2.jpg" (во втором случае файл "2.jpg" должен располагаться в папке с проектом)
Название таблицы Имя таблицы в базе данных
