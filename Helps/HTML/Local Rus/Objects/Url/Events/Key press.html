Описание

Событие происходит у активного сейчас объекта (объекта в фокусе), в момент, когда пользователь нажимает кнопку клавиатуры. У объекта, на котором произошло данное событие, появляются дополнительные свойства.

Дополнительные свойства, которые появляются у объектов при данном событии:

Объекту, с которым произошло данное событие, добаляется рад новых свойств (например, X и Y курсора мыши, нажатая клавиша). Эти свойства нельзя изменять, но можно их посмотреть (т.к. их нельзя менять, то найти эти свойства можно в Мастере сложных действий). Ниже перечислены все свойства данного события.

Свойство
Описание
_Введенный символ Буквенный символ (а, б, 3), который ввели с клавиатуры в данное событие.
_Отменить вводДанное свойство позволяет отменить введенный пользователем символ. Делается это изменением значения свойства на "Да".
