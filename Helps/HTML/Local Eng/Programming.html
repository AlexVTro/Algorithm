Description

The development environment of Algorithm2 has a lot of complex means for programming.

Debugging You can easily control a program when it is running in the Development Environment of Algorithm2. You can suspend a program at any action, look the value of some property or set another action.
Quick fill wordsQuick fill words are letter symbols for specifying of different values (colors, cursors, etc.). They are used when it is necessary to assign a the property the value.
