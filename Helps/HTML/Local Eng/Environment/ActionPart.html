Description

Actions are commands (change a text of the window, change button color) executed by the program and programmed by you.

You can set an action for the object using the Actions tab of Algorithm2 development environment:

There you should specify the object intended for executing of your action and select a property which is to be changed. Below you should specify a new value for this property and then press the button Create. In such a way you'll create an action:

So when the object Button has "received" the event Click all actions inside of this event are executed. In this example the text of the window title will change to "Window1"

Remark

New values of actions can be not only simple (text, numbers), but also complex (formula, a property value of some other object). To make a complex action use Complex operations wizard (click on the button with a magic wand on it)
