Description

Properties panel helps to set all necessary properties of main elements - objects (buttons, labels, windows).

Properties are settings of objects (color, text, size) by changing which you can customize the object in a necessary way. Learn more details about properties of each object in the section Objects.

To edit property select it in the list, edit a value and then press Enter.
