Description

Window is a panel where you project graphic design of your program by placing different objects in the area of the window panel (buttons, text boxes, tables).

All objects can be relocated in any way or brought to the front. You can also change sizes, create arrays and so on.

The number of windows is unlimited. For convenience each new window occupies a separate tab.

Some objects (memory cells, a stopwatch, print dialog) are located not in the window panel, but on the panel below. These are not graphical objects, but software objects. Such objects are hidden when the program works as they are designed for auxiliary operations (e.g. data storage, time reckoning, printing)
