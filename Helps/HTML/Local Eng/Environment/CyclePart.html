Description

Cycles are same conditions except for that all actions inside of cycles are being executed constantly, until the condition is false (e.g. until the height of the window is less than 100)

You can create a cycle in the Cycles tab of Algorithm2 development environment:

There you should specify the condition of cycle itself in the form of: "something" is equal/unequally to "something" ("unequally" is specified by the symbol <>, "more or equal" - by the symbol >=) 

Window1.Text1.Text = "Required text"
Window1.Window1.Width > "200"
Window1.Button1.Fore color  <> black
Window1.Window1.X >= Window1.Text1.Text
After you have clicked the button Create cycle block appears in the Actions tree.

Now all actions inside of this cycle block are being constantly cyclically executed until the cycle condition is false. In other respects cycle conditions function the same way standard conditions function.

Remark

Cycle conditions can be not only simple (texts or numbers comparison), but also complex (formula, comparing with a property value of the other object). To create a complex condition use Complex operations wizard (click on the button with a magic wand on it).

You can interrupt a cycle pre-term, without waiting for the rest of actions have been executed. Learn more about it in the section Useful Objects.Interrupts
