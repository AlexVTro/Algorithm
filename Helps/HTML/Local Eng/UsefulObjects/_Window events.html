Description

The useful object Window events makes accessible the properties and commands of thatwindow, or of that object of this window where the event is being executed at the moment.

Example:

There Window events makes accessible properties and commands of the Window1, because the event is executed under the Button1 and this button locates in the Window1.

Thereby Window events has nothing new, it just simplifies the process of actions creation. Now in case it is necessary to change the property of the window where the event has been executed, there is no need to specify its name (e.g. Window1), but simply the universal name "Window events" can be specified.

Features of the object:
Properties and Commands Properties are settings of objects (color, text, size). By changing them you can set the object in a way you need. Objects can also execute some actions which are called commands and relate to them. You can change properties manually in Properties panel or program their changes in Actions tab.
