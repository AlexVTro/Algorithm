Description

The property is used to find out whether a certain table cell is selected or to select a cell.

The data required to implement properties or commands:

Objects._Object events.Item selected (Line numbers separated by commas, Column numbers separated by commas)

Data
Description
Line numbers separated by commas Перечислите нужные строки через запятую в двойных кавычках, например, "0,1,8,3" или "2".
Column numbers separated by commas Здесь требуется указать столбцы, с которыми будут производиться действия. Если столбец один, то можно укать так: "0" - первый столбец или "2" - третий. Если несколько, то: "0,1,3" - первый второй и четвертый столбец.
