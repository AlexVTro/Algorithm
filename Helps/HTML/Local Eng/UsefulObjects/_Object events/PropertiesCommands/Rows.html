Description

Rows are data recorded in the table. Rows are specified in the form of "Row1" | "Row1", "Row2" | "Row2" and divided with commas (","). So the first row of the table is - "Row1" | "Row1", the second row is "Row2" | "Row2". Cells inside of one row are divided with the symbol ("|"), i.e. a row "Row1" | "Row1" consists of two cells and every cell contains Row1.

Remark

Quotes in the record "Row1" | "Row1", "Row2" | "Row2" are obligatory. If you want to change columns in the process of program work, then the action should be as follows: 
Window1.Table1.Rows = """Row1"" | ""Row1"", ""Row2"" | ""Row2"""
As two double quotes ("") inside of the text are converted into one double (") we'll get eventually: "Row1" | "Row1", "Row2" | "Row2"
