Description

Properties are settings of objects (color, text, size). By changing them you can set the object in a way you need. Objects can also execute some actions which are called commands and relate to them.

You can change properties manually in Properties panel or program their changes in Actions tab. In the second case in the Events tab the event is created for the object and under this event the given property is changed. Examples of how to use properties and commands in Actions tab:

Window1.Object1.Text = "New text1"
Window1.Object2.Image = "C:\Pic.jpg"
Window1.Object2.Focus

List of object properties _Text:
_Chars The property returns a character in accordance with the ordinal number specified in the property. A first character has the ordinal number "1". If there are less characters in the text than the number you've specified, this property returns nothing, i.e. "".
_Chars length This property returns the number of characters in the text.
_Compare The command is used for comparing texts and in case they are identical it returns "0". In case they are non-identical the command returns "-1" when the first text is prior to the second text in alphabetical order. If the second text is prior in alphabetical order the command returns "1". Thus with the help of this command you can, for example, sort texts alphabetically.
_Consist The command searches any specified characters in the initial text. If even one character is found, the command returns the ordinal number of this character. For example, if to search "t" in the text "Algorithm is a good program", the command would return "7", i.e. the number of the character in the text.
_Consist all Confirms that the text consists purely of characters specified for this command. If even one character differs, the command returns "No".
_Consist no The command searches any specified characters in the initial text. If even one character is found, the command returns "No". If there are no specified characters in the text, the command returns "Yes". For example, if in the text "Algorithm is a good program" it is necessary to check the presence of characters "job8", the command would return "No" (because the text contains a character "o").
_Decrypt text
_Delete quotes The command deletes quotes if the initial text was put in them. If there are two double quotes ("") inside of the text, they would become one double quote ("" -> "). E.g. the text "Hi ""friend""" after applying of this command would turn into Hi "friend".
_Encrypt text
_Index of The command searches the specified text and returns the number of a character the text starts from. Letter case is ignored while searching, i.e. letters "A" and "a" are regarded as identical. In case nothing is found the command returns "0".
_Index of ignore case The command searches the specified text and returns the number of that character from which the text starts. In case nothing is found the command returns "0". While searching the letter case is ignored, i.e. letters "А" and "а" are regarded as identical.
_Index of line
_Index of regular The command searches in the text a specified pattern of regular expression (e.g. "Image1\.(jpg|gif|png)") and returns the number of character the string starts with. In case nothing was found the command returns "0". How to compose regular expressions learn there: http://en.wikipedia.org/wiki/Regexp.
_Index of without quotes The command searches the specified text and returns the number of character the text starts with. Letter case is ignored while searching, i.e. letters "A" и "a" are regarded as identical. If while searching a piece of text in quotes is found then this piece is ignored. In case nothing is found command returnns "0".
_InsertInserts whatever you specify in any part of the text.
_Is number text Checks whether a text is a number or not ("23", "-12,54", "3,2+E12").
_Is numeric text Checks whether the text consists exclusively of figures ("0123456789") or not.
_Last index of ignore case The command searches the specified string from the end of the text and returns the number of a character the string starts with. Letter case is ignored while searching, i.e. letters "A" and "a" are regarded as identical.
_Line This property returns the string according to the ordinal number specified in the property. The first string has the number "1". If there are less strings in the text than the number specified, the property returns nothing, i.e. "".
_Lines count
_Put in quotes The command puts the given text in quotes. In case there is a double quote (") in the text it converts it into two double quotes (" -> ""). For example, a text Hi "friend" will turn into "Hi ""friend""".
_RemoveRemoves a certain part of the text from it.
_Replace allIn accordance with a pattern it searches in the text all strings and replaces all them by specified texts.
_Replace oneIt searches the specified string in the text, returns the first appropriate string and replaces it with specified text.
_Split It is used to split the text into parts divided with some symbol or a line. It returns the specified part of the text. For example, if you split the text "Hi! How do you do?" into parts with "!" as a splitter then you'll get "Hi" and " How do you do?". If you specify 2 as a number of the part you'll get " How do you do?".
_Split count Is used to know into how many parts the text can be split when its parts are divided by a symbol or a line. For example, if to split the text "Hi! How do you do?" into parts with "!" as a splitter then the text will be split into "Hi" and "How do you do?" and the command will return the number of parts - "2".
_Split without quotes It is used to split any text into parts divided with some symbol or a line and returns the specified part of the text. If a part of text in quotes is found while searching then it ignores this piece and no searching inside of this piece is done.
_Split without quotes count It is used to know on how many parts the text can be split in case its parts are divided with some symbol or line. In case a part of the text in quotes is found while searching, this part is ignored and searching is not done inside of it.
_Substring It is used to get a part of the text if the number of the part and its length are known. For example, from the text "Hello World!" you can get the second word ("world") specifying that the part starts from the 7th character and the length is 5 characters.
_To lower The command converts all letters into lower case ("А" -> "а").
_To upper The command converts all letters into upper case ("а" -> "А").
_Trim all Removes all symbols (space, tab, paragraph) from the end and beginning of the text.
_Trim end Removes all spacing symbols (space, tab, paragraph) from the end of the text.
_Trim start Removes all spacing symbols (space, tab, paragraph) from the beginning of the text.
