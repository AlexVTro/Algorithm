Description

It is used to split the text into parts divided with some symbol or a line. It returns the specified part of the text. For example, if you split the text "Hi! How do you do?" into parts with "!" as a splitter then you'll get "Hi" and " How do you do?". If you specify 2 as a number of the part you'll get " How do you do?". Often it is necessary to know how many parts a text is split in. You can get it using the command _Split count.
