Description

The command searches in the text a specified pattern of regular expression (e.g. "Image1\.(jpg|gif|png)") and returns the number of character the string starts with. In case nothing was found the command returns "0". How to compose regular expressions learn there: http://en.wikipedia.org/wiki/Regexp.

The data required to implement properties or commands:

_Useful objects._Text._Index of regular (Original text, Regular Expression Pattern)

Data
Description
Original text Текст оригинал, который будет использоваться данным свойством или коммандой.
Regular Expression Pattern Шаблон, заданный в синтаксисе языка регулярных выражений. Об этом языке можно почитать здесь: http://ru.wikipedia.org/wiki/Регулярные_выражения. Например, шаблон "Рисунок1\.(jpg|gif|png)" соответствует "Рисунок1.jpg","Рисунок1.gif", но не подходит для "Рисунок1.jpeg" или "Рисунок1gif".
