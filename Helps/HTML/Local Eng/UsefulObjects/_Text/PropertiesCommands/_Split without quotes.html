Description

It is used to split any text into parts divided with some symbol or a line and returns the specified part of the text. If a part of text in quotes is found while searching then it ignores this piece and no searching inside of this piece is done.

Foe example, if we split the text Hi! Friend"!" How do you do? into parts with "!" as a splitter, then the text will be spliy into 'Hi' and 'Friend"!" How do you do?'. And if you specify 2 as the number of the part И you'll get 'Friend"!" How do you do?''. Often it is necessary to know about how many parts the text was split it. You can get it using the command _Split without quotes count.

The data required to implement properties or commands:

_Useful objects._Text._Split without quotes (Original text, The symbol of the division of, The number of correct)

Data
Description
Original text Текст оригинал, который будет использоваться данным свойством или коммандой.
The symbol of the division of Символы, которые будут разделять части текста. Например ",", "; ", "а", " и ", и т.д.
The number of correct После того как текст разобьется на части, нужно указать какую часть из них взять. Первая часть имеет номер "1".
