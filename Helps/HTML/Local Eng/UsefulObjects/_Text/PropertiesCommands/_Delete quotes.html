Delete quotes
Help -> Useful Objects -> _Text -> Properties and commands -> _Delete quotes

See also
Collapse all

Description

The command deletes quotes if the initial text was put in them. If there are two double quotes ("") inside of the text, they would become one double quote ("" -> "). E.g. the text "Hi ""friend""" after applying of this command would turn into Hi "friend".
