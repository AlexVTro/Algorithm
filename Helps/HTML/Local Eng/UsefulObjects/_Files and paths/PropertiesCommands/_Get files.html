Get files
Help -> Useful Objects -> _Files and paths -> Properties and commands -> _Get files

See also
Collapse all

Description

The property returns a list of files from the folder specified. Files in the list are enumerated in "; " and it looks like: "C:/1.jpg; C:/2.bmp; C:/file.exe".

The data required to implement properties or commands:

_Useful objects._Files and paths._Get files (The path to the file or folder)

Data
Description
The path to the file or folder Тут нужно указать путь к файлу или папке. Путь должен обозначаться в кавычках, например "C:\1.jpg" или "\2.jpg" (во втором случае файл "2.jpg" должен располагаться в папке с проектом).
