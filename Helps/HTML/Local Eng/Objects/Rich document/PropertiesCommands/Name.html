Description

All objects in Algorithm2 are addressed by names. Using this property you can unambiguously specify an object which you want to work with.

An object can have absolutely any name, the main thing is that you should exactly know what object is at the back of the name.

Remark

In fact several objects can have same names, but they must have different indices. In this case objects are united into a useful construction which is called Array.
