Description

Events are actions performed by the user (clicking, cursor moving, keys pressing) which can be registered by objects.

Every object has particular events to register. You can program registering of events in the Actions tab. After you've programmed it all actions inside of the event in the Actions tree would be executed only when the given event has occured (for example, after clicking).

ClickThis event is called when the user clicks an object with a mouse.
Created The event is called when the program is started - when the object is created for the first time.
Download cancelledThe event is called when the user canceles file downloading.
Paint The event is called when the object is completely or partially redrawn. This occurs when the object was hidden under some other object and now becomes visible. Or this can occur after the cursor was moved over the object and then left the object's area and the object is redrawn in order to delete cursorimage.
Receive progressThe event is called while data is received.
Received responseThis event is called immediately after the server has responded to your request.
Sending queryThe event is called while the request is sent to the server.
Sent queryThe event is called immediately after the request has been sent to the server.
