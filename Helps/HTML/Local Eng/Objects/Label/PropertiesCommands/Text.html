Text
Help -> Objects -> Label -> Properties and commands -> Text

See also, Sample
Collapse all

Description

A text specified in this property is displayed on the object.

For example, the text of the object Window is its title and the text of the object Text box is a text which is input in it.

Remark

If it is necessary to input quotes into text then you should double them inside of the text:
"There a text is without quotes and ""there is in quotes""."

To insert Newline (equal to pressing of Enter key) use Quick fill words -> Others or input a symbol "¶"

Window1.Button1.Text = "Text ¶ Text on a newline"

Example

Variants of how main window text can be set:

Algorithm2
[Copy to clipboard] 
Window1.Window1.Text = "New text of window title"
Window1.Window1.Text = "New" & "text of window title"
Window1.Window1.Text = Window1.Window1.Text & "<- window text"
Window1.Window1.Text = "Window width: """ & Window1.Window1.Width & """"
