Description

Along with name index is an identifying property of the object. This property helps to differentiate objects with same names (in arrays).

Index can be any positive number. Besides, when you create actions and it is necessary to specify the object index, you can use Quick fill word "All...". In this case the created action is applied to all objects with the specified name independently of their indices.

Remark

Two objects should not have both same names and indices, because these properties are destined for unambiguous identification of the object in the Algorithm environment.
