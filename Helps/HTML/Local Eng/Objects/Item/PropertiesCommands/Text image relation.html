Text image relation
Help -> Objects -> Item -> Properties and commands -> Text image relation

See also, Sample
Collapse all

Description

It defines position of text on the object relative to the image (text is in front of the image or on the left of the image, etc.).

All available text and image positions are described in Quick fill words -> Text image relation.

Example

Variants of how button text and image position can be set:

Algorithm2
[Copy to clipboard] 
Window1.Button1.Text image relation = "On top"
Window1.Button1.Text image relation = On left
Window1.Button1.Text image relation = Window1.Button2.Text image relation
Window1.Button1.Text image relation = Window1.Text1.Text
