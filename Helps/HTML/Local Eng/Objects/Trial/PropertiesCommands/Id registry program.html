Description

ID registry program defines where in the registry data about trial period of your program should be stored. All registration keys are valid for the paticular ID program. If you'd change it, the keys would become invalid for the new ID program. And encryption key should also remain the same.

If you use two objects Trial, for example, one for activation key input and the other for generating these keys (like in the example Examples\Difficult projects\Trial) then make sure that Id program and encryption key are the same!
