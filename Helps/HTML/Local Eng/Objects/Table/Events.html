Description

Events are actions performed by the user (clicking, cursor moving, keys pressing) which can be registered by objects.

Every object has particular events to register. You can program registering of events in the Actions tab. After you've programmed it all actions inside of the event in the Actions tree would be executed only when the given event has occured (for example, after clicking).

Created The event is called when the program is started - when the object is created for the first time.
Double clickThe event is called when an object is quickly doubly clicked.
Got focus The event is called when the object gets focus (it becomes active).
Key down The event is called for the active object (the object is in focus) when the user presses a keyboard key. This object gets additional options.
Key press The event is called for the active object (the object is in focus) when the user releases a keyboard key. Then this object gets additional options.
Key up The event is called for the active object (the object is in focus) when the user presses a keyboard key. Then this active object gets additional options.
Lost focus The event is called when the object loses focus (i.e. some other object becomes active).
Mouse down The event is called for the active object (the object is in focus) when the user clicks a mouse button. As a result the object gets additional options.
Mouse enterThe event is called as soon as the cursor is moved over the object.
Mouse hoverThe event is being constantly executed until the cursor is out of the object's area.
Mouse leaveThe event is called as soon as the cursor leaves the object's area.
Mouse moveThe event is called when the user moves the mouse in the object's area. As a result the object gets additional options.
Mouse up The event is called when the user releases pressed mouse button. As a result the object gets additional options.
Mouse wheel The event is called for the active object (the object is in focus) when the user begins to use a mouse wheel. As a result this active object gets additional options.
Paint The event is called when the object is completely or partially redrawn. This occurs when the object was hidden under some other object and now becomes visible. Or this can occur after the cursor was moved over the object and then left the object's area and the object is redrawn in order to delete cursorimage.
