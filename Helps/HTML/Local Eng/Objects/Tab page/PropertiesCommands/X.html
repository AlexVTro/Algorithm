Description

Coordinate X of the object's upper left edge.

Coordinates of all objects are arranged not like usual coordinates. The origin of coordinates is not in the left bottom edge, but in the left upper one. So the coordinate X is nothing but a distance from the left edge.

Remark

If you placed an object inside of the other object (e.g. inside of a panel), then coordinates are calculated relative to this object (panel).

Example

Variants of how button coordinate X can be set:

Algorithm2
[Copy to clipboard] 
Window1.Button1.X = 150
Window1.Button1.X = -10
Window1.Button1.X = Window1.Window1.Width / 2
