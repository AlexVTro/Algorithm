Description

The main function of the object is to display the text with url after clicking which the user jumps, for example, to the web page.

Urls can refer not only to the Internet. After you've set the event Link clicked the click on the specified link would be processed in a way you need.

Features of the object:
Events Events are actions performed by the user (clicking, cursor moving, keys pressing) which can be registered by objects.
Properties and Commands Properties are settings of objects (color, text, size). By changing them you can set the object in a way you need. Objects can also execute some actions which are called commands and relate to them. You can change properties manually in Properties panel or program their changes in Actions tab.
