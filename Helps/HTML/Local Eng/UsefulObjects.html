Description

Useful Objects (date, files and paths, registry) function like standard objects, but they have no graphic interface and are designed for interaction with the OS "Windows" (in order to open a file, get date and time or cursor coordinates).

Useful Objects extend substantially the possibilities of working in Algorithm2. Using them you can get to the core of the OS (for example, you can change Wall paper of the desktop). Moreover, they help to simplify some calculations (e.g. calculations of dates).

The list of all Useful Objects:
_Advanced toolsThis useful object is the most advanced tool available for you in the environment of Algorithm2. It executes any programme code written in high-level programming languages - VB.NET, C#, VBScript. It also executes the programme code of Algorithm2.
_Call event Using this useful object you can call any event of the program (click, text input, mouse movement) independently of the user actions.
_DateThe object is used for calculations relating to dates and time. Using the object you can calculate, for example, the day of the week, the current month or how much time has elapsed from the specific moment.
_Files and pathsThe object is used for any manipulations with files and paths.
_InterruptsAs is obvious from the title, this object is used to interrupt execution of the program (or events, conditions or cycle). To interrupt means not to execute actions which follow the command of interruption.
_Object eventsThe useful object Object events makes accessible the properties and commands of that object where the event is being executed at the moment.
_RegistryThe object gives an access to the registry of the operational system Windows. You can edit it, create or delete keys and so on.
_ScreenThe object makes accessible the properties and commands of the computer screen and desktop.
_Show messangeThe object is used to display Windows standard messages. It also helps to register which button of the message the user has clicked (OK, Cancel, Abort).
_SystemThe object gives an access to the holy of holies - to OS Windows. You can control a mouse, restart the computer, start any applications and more.
_TextThe object is used for working with texts and implements such options as text search, replacement, partition of the text, etc.
_Window eventsThe useful object Window events makes accessible the properties and commands of thatwindow, or of that object of this window where the event is being executed at the moment.
